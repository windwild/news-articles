# More ways to work with your team and tools in ChatGPT

# 在 ChatGPT 中以更多方式与团队及工具协同工作

New shared projects, smarter connectors, and compliance and security updates help teams get more done.

新增的共享项目、更智能的连接器，以及合规性与安全更新，助力团队提升工作效率。

![Minimalist UI icons representing sharing, user folders, and security (padlock), set against a soft blue gradient background.](images/more-ways-to-work-with-your-team-and-tools-in-chatgpt-openai/img_001.png)

We want to help teams and organizations do their best work with ChatGPT. Today we’re rolling out shared projects for [ChatGPT business⁠(opens in a new window)](https://chatgpt.com/business?openaicom-did=11fc5d2b-0e40-40ab-a7d6-3e7ab5490007&openaicom_referred=true) plans; we’ve also recently added new connectors to team tools and improved response speed and accuracy, along with new compliance and admin features:

我们希望助力团队和组织借助 ChatGPT 发挥最佳工作效能。今天，我们正式面向 [ChatGPT Business 计划⁠(在新窗口中打开)](https://chatgpt.com/business?openaicom-did=11fc5d2b-0e40-40ab-a7d6-3e7ab5490007&openaicom_referred=true) 推出共享项目功能；同时，我们近期还为团队常用工具新增了多项连接器，并提升了响应速度与准确性，此外还推出了多项新的合规性与管理功能：

- **Shared projects help teams move faster**: Teams can now work with ChatGPT toward a goal—whether supporting a customer account, creating content, or running monthly reports—while staying on the same page and keeping work consistent. Members can add files and instructions to a shared project together, and ChatGPT uses that context to tailor its responses so every new chat starts with the latest information.

- **共享项目助力团队加速推进工作**：团队现可围绕统一目标（例如支持客户账户、创作内容或生成月度报告）协同使用 ChatGPT，在保持信息同步的同时确保工作成果的一致性。团队成员可共同向共享项目中添加文件与指令，ChatGPT 将基于该上下文定制化响应，确保每一次新对话均以最新信息为起点。

- **More helpful responses from your team’s tools**: With new connectors, ChatGPT can pull information from Gmail, Google Calendar, Microsoft Outlook, Microsoft Teams, SharePoint, GitHub, Dropbox, and Box for more relevant answers based on your work. Responses are now faster and more accurate, and ChatGPT can decide when to use the right connector for each prompt.

- **来自团队工具的更实用响应**：借助新增连接器，ChatGPT 可从 Gmail、Google 日历、Microsoft Outlook、Microsoft Teams、SharePoint、GitHub、Dropbox 和 Box 等工具中提取信息，从而基于您的实际工作场景提供更相关、更精准的答案。当前响应速度更快、准确性更高，且 ChatGPT 能够自主判断并为每个提示词选择最合适的连接器。

- **Compliance and admin features keep work data secure**: New ISO 27001, 27017, 27018, and 27701 certifications, an expanded SOC 2 report, role-based access controls, and enhanced SSO help organizations confidently enable every employee with AI.

- **合规性与管理功能保障工作数据安全**：新增 ISO/IEC 27001、27017、27018 和 27701 认证，扩展后的 SOC 2 报告，基于角色的访问控制（RBAC），以及增强型单点登录（SSO）功能，助力组织放心地为每一位员工赋能 AI。

## Shared projects help teams get more done together

## 共享项目助力团队高效协同

Last year we launched projects to keep related chats, files, and instructions in one place so you can pick up work where you left off without re-explaining context to ChatGPT. Now you can share projects with teammates, adding files and instructions together to guide ChatGPT responses toward a shared goal. Members can chat with the project’s context to stay on the same page as new information gets added and create work that stays consistent in tone and style.

去年，我们推出了“项目”功能，将相关对话、文件与指令集中管理，让您无需反复向 ChatGPT 解释上下文即可无缝续接先前工作。如今，您可将项目与团队成员共享，共同添加文件与指令，引导 ChatGPT 围绕统一目标生成响应。团队成员可在项目上下文中开展对话，确保随着新信息持续加入，所有人始终步调一致，并产出风格与语调高度统一的工作成果。

#### Ways to use shared projects

#### 共享项目的典型应用场景

In early testing, teams used them to coordinate:  

在早期测试阶段，团队已将其用于协调以下工作：

- **Client work:** Keep all notes, proposals, and contracts in one place so account teams can draft follow-ups faster and stay aligned. For example, as an account manager, you can upload call notes and your solutions engineer can ask ChatGPT to summarize the latest updates to prepare for a customer demo.  
- **客户工作：** 将所有会议纪要、提案和合同集中存放在一处，以便客户团队能更快起草后续跟进内容，并始终保持步调一致。例如，作为客户经理，您可以上传电话会议纪要；而您的解决方案工程师则可让 ChatGPT 汇总最新进展，为面向客户的演示做好准备。

- **Content creation:** Use project-wide instructions to maintain a consistent voice and style. You can, for instance, tell ChatGPT to follow your brand guidelines so project members can create on-brand assets, from blogs to social copy, for an upcoming product launch.  
- **内容创作：** 利用项目级指令，确保整体语气与风格统一。例如，您可指示 ChatGPT 遵循您的品牌规范，使项目成员能为即将发布的新产品，高效产出符合品牌调性的各类内容——从博客文章到社交媒体文案。

- **Reporting:** Split tasks while staying coordinated with shared instructions and uploaded datasets. Teams preparing monthly reporting can prepare updates for different audiences, such as an executive summary or a more detailed breakdown for finance.  
- **报告编制：** 在共享指令与已上传数据集的支持下合理分工、协同推进。例如，负责月度报告的团队可分别面向不同受众（如高管层的摘要版，或财务部门所需的详细分析版）同步开展更新工作。

#### How it works  
#### 工作原理  

Project creators can invite teammates by email or link with two levels of access, chat or edit. Chat access lets members see and interact with the project’s chats, files, and instructions; edit access adds the ability to update instructions, upload or remove files, and invite others.  
项目创建者可通过电子邮件或链接邀请团队成员加入，权限分为两级：“聊天”（Chat）与“编辑”（Edit）。“聊天”权限允许成员查看并参与项目内的对话、文件及指令；“编辑”权限在此基础上，额外赋予更新指令、上传或删除文件、以及邀请其他成员的权限。

Shared projects have their own private memory, which lets ChatGPT remember details members share so you can start new conversations without explaining everything again. This is especially useful for long-running work with multiple follow-ups, and keeps sensitive data or client details safely within the project.  
共享项目拥有独立的私有记忆空间，使 ChatGPT 能记住成员在项目中分享的关键信息，从而让您开启新对话时无需反复解释背景。这一功能对周期较长、需多次跟进的工作尤为实用，同时也能确保敏感数据或客户信息安全地保留在项目内部。

Learn more in our [Help Center.⁠(opens in a new window)](https://help.openai.com/en/articles/10169521-projects-in-chatgpt#h_e8f291686b)  
更多详情，请参阅我们的 [帮助中心。（在新窗口中打开）](https://help.openai.com/en/articles/10169521-projects-in-chatgpt#h_e8f291686b)

**What’s next**: In this first version, project members can work asynchronously, with the ability to build on each other’s updates, start their own one-on-one conversations with ChatGPT, or branch another member’s conversation to explore a new idea. This release is an early step toward team collaboration in ChatGPT, and we’ll use your feedback to shape what comes next.  
**后续计划：** 在当前首个版本中，项目成员可异步协作——既能基于彼此的更新持续推进工作，也可各自启动与 ChatGPT 的一对一对话，甚至可“分叉”（branch）其他成员的对话，以探索全新思路。本次发布是 ChatGPT 迈向团队协作的重要初步尝试；我们将持续收集您的反馈，用以规划后续功能演进。

**Availability**: Shared projects are available to Business, Enterprise, and Edu plans today, with Free, Go, Plus, and Pro coming soon. They are defaulted off for Enterprise and Edu customers, and admins can control access. Projects can only be shared with workspace members for ChatGPT business plans.  
**可用性：** 共享项目目前已面向 ChatGPT Business、Enterprise 及 Edu 订阅计划开放；Free、Go、Plus 与 Pro 计划将于近期上线。对于 Enterprise 和 Edu 客户，该功能默认处于关闭状态，管理员可自主控制访问权限。此外，针对 ChatGPT Business 计划，项目仅可与同一工作区（workspace）内的成员共享。

## Smarter answers from your team’s tools  
## 来自您团队工具的更智能回答  

Teams generate large amounts of documents, chats, and emails, so finding the right information when you need it can be hard. With connectors, ChatGPT pulls in relevant context from tools your team uses every day, like Google Drive, SharePoint, GitHub, Microsoft Teams, or Gmail, so you can onboard to a new codebase, write a marketing brief using the latest template, or summarize important emails. It’s like working with a teammate who gets your work and can give expert answers on almost any topic.  
团队日常会产生海量文档、聊天记录和邮件，因此在需要时快速定位准确信息往往颇具挑战。借助连接器（connectors），ChatGPT 可自动从您团队日常使用的各类工具（如 Google Drive、SharePoint、GitHub、Microsoft Teams 或 Gmail）中提取相关上下文——助您快速熟悉新代码库、依据最新模板撰写营销简报，或汇总关键邮件要点。这就像拥有一位深谙您业务的同事，几乎能在任何主题上为您提供专业级解答。

Recent updates make connectors even more useful: responses are faster and more accurate and ChatGPT now knows when to use connectors automatically. New email and calendar connectors let you pull context from upcoming meetings and recent conversations so you can draft agendas, follow up on action items, or quickly find details without switching apps.

近期更新让连接器（connectors）变得更加实用：响应速度更快、准确性更高，且 ChatGPT 现在能够自动判断何时启用连接器。全新的电子邮件与日历连接器可从即将召开的会议及近期对话中提取上下文，助您快速拟定议程、跟进待办事项，或无需切换应用即可迅速查找到关键信息。

#### Ways to use connectors

#### 连接器的多种使用方式

How teams use connectors today:

当前团队如何使用连接器：

- **Communication and writing:** Draft updates, synthesize documents, or follow guidelines and templates (e.g., “Using our web template in Google Drive, create a brief for \[x\]”).

- **沟通与写作**：草拟工作简报、整合多份文档，或遵循既定指南与模板（例如：“使用 Google Drive 中的网页模板，为 \[x\] 创建一份简报”）。

- **Meetings and collaboration:** Prepare for meetings, create agendas, and identify action items (e.g., “Review my latest emails with \[customer/company\] to prep for our next call”).

- **会议与协作**：会前准备、制定会议议程、识别待办事项（例如：“查阅我最近与 \[客户/公司\] 的邮件往来，为下一次通话做准备”）。

- **Organization and productivity:** Quickly find important documents or answers in company data (e.g., “How do I submit an expense report?”).

- **组织与提效**：在企业数据中快速定位重要文档或问题答案（例如：“我该如何提交差旅报销？”）。

- **Analysis and reporting:** Analyze data for market research or performance tracking (e.g., “Tell me which campaigns drove our highest-value deals in HubSpot”).

- **分析与报告**：开展市场调研或业绩追踪类数据分析（例如：“告诉我，在 HubSpot 中哪些营销活动促成了我们价值最高的成交订单？”）。

- **Coding and technical tasks:** Understand codebases, review pull requests, or write documentation (e.g., “How is \[feature\] implemented in our codebase?”).

- **编程与技术任务**：理解代码库结构、评审 Pull Request，或编写技术文档（例如：“\[功能\] 在我们的代码库中是如何实现的？”）。

#### Recent updates

#### 最近更新

**Added more productivity tools:** New connectors to Gmail, Google Calendar, Google Contacts, Microsoft Outlook and Calendar, and Microsoft Teams give ChatGPT a more complete view of projects, communications, and schedules, so teams spend less time chasing information across tools.

**新增更多效率工具连接器**：新增对 Gmail、Google 日历、Google 联系人、Microsoft Outlook 与日历、以及 Microsoft Teams 的连接支持，使 ChatGPT 能更全面地掌握项目进展、沟通动态与日程安排，从而大幅减少团队在不同工具间反复查找信息所耗费的时间。

**ChatGPT knows when to use connectors:** We’ve improved ChatGPT’s ability to automatically decide when to use connectors, search the web, or rely on its own training—making your company context a more seamless part of the conversation. It can also pick the most relevant connector based on the question, so you no longer need to turn individual connectors on in every new chat.

**ChatGPT 能自主判断何时启用连接器**：我们进一步提升了 ChatGPT 自动决策能力——它能智能判断何时调用连接器、何时进行网页搜索、或仅依靠自身训练知识作答，从而使您的企业上下文更自然、更无缝地融入对话之中。此外，它还能根据提问内容自动选择最相关的连接器，因此您无需再在每一次新对话中手动开启特定连接器。

**Faster, more accurate responses**: ChatGPT can now sync data from GitHub, SharePoint, Dropbox, and Box ahead of time, so it responds faster and more accurately. Learn more about [synced connectors⁠(opens in a new window)](https://help.openai.com/en/articles/10847137-chatgpt-synced-connectors).

**响应更快、更精准**：ChatGPT 现在可预先同步 GitHub、SharePoint、Dropbox 和 Box 中的数据，从而实现更快速、更准确的响应。了解有关 [已同步连接器⁠（在新窗口中打开）](https://help.openai.com/en/articles/10847137-chatgpt-synced-connectors) 的更多信息。

**What’s next**: In the coming weeks, we’ll roll out new connectors for popular team tools across CRM, communication, knowledge stores, and ticketing, bringing ChatGPT closer to everything your business runs on. Soon, we’ll also make connectors available in projects and GPTs.

**下一步计划**：在未来几周内，我们将陆续推出面向 CRM、通信、知识库及工单系统等主流团队工具的新连接器，让 ChatGPT 更深度融入您企业日常运行的每一环节。不久之后，连接器功能还将扩展至“项目（Projects）”和“GPTs”中。

**Availability**: Connectors are now generally available and can be enabled in [Settings⁠(opens in a new window)](https://chatgpt.com/?openaicom-did=11fc5d2b-0e40-40ab-a7d6-3e7ab5490007&openaicom_referred=true#settings/Connectors). Admins control user access for ChatGPT business plans, and they’re off by default for Enterprise and Edu. As always, connectors respect existing content permissions, so ChatGPT only sees what each person is allowed to access, and all data remains encrypted in transit and at rest. We never train on content shared with ChatGPT for our business plans by default [⁠⁠(opens in a new window)](https://help.openai.com/en/articles/7730893-data-controls-faq). See our [Help Center⁠(opens in a new window)](https://help.openai.com/en/articles/11487775-connectors-in-chatgpt) for details on availability by plan.

**可用性**：连接器现已全面开放，并可在 [设置⁠（在新窗口中打开）](https://chatgpt.com/?openaicom-did=11fc5d2b-0e40-40ab-a7d6-3e7ab5490007&openaicom_referred=true#settings/Connectors) 中启用。ChatGPT 商业版计划的用户访问权限由管理员统一管控；而企业版（Enterprise）与教育版（Edu）默认关闭该功能。一如既往，所有连接器均严格遵循现有内容权限体系——ChatGPT 仅能访问每位用户被授权查看的内容；所有数据在传输和静态存储过程中均保持加密。对于我们的商业版计划，默认情况下，我们绝不会使用用户分享给 ChatGPT 的内容进行模型训练 [⁠⁠（在新窗口中打开）](https://help.openai.com/en/articles/7730893-data-controls-faq)。有关各版本计划中连接器功能的具体可用性，请参阅我们的 [帮助中心⁠（在新窗口中打开）](https://help.openai.com/en/articles/11487775-connectors-in-chatgpt)。

## New compliance programs and admin controls keep company data secure

## 新增合规计划与管理员控制功能，保障企业数据安全

As we make ChatGPT more valuable for the workplace, we’re equally focused on helping organizations deploy it across their company responsibly and with confidence. To support this, we’ve added new compliance and admin features that give companies more control over who can access what, where their data lives, and how it’s protected.

随着 ChatGPT 在职场中日益发挥更大价值，我们同样致力于助力各组织负责任且自信地在全公司范围内部署该工具。为此，我们新增了多项合规性与管理员管控功能，使企业能够更精细地掌控：谁能访问哪些资源、数据存于何处，以及如何确保其安全。

#### Recent updates

#### 最近更新

**Compliance certifications:** OpenAI’s Information Security Management System is now certified by Schellman to ISO/IEC 27001, 27017, 27018, and 27701 standards. We’ve also expanded our SOC 2 compliance to cover four Trust Services Criteria, now including Security, Confidentiality, Availability and Privacy. Customers can view our [ISO certificate⁠(opens in a new window)](https://trust.openai.com/?a9437b=a9437b&itemUid=5c93ca3e-519a-4260-a50a-a9de78294538&source=click) and [SOC 2 report⁠(opens in a new window)](https://trust.openai.com/?itemUid=fa950d02-cbb3-4010-b917-7137a7c2a982&source=click) on our Trust Portal.

**合规认证**：OpenAI 的信息安全管理体系（ISMS）现已获得 Schellman 公司认证，符合 ISO/IEC 27001、27017、27018 及 27701 国际标准。同时，我们的 SOC 2 合规范围也已扩展至涵盖全部四项可信服务准则（Trust Services Criteria），即安全性（Security）、保密性（Confidentiality）、可用性（Availability）与隐私性（Privacy）。客户可通过我们的信任中心（Trust Portal）查阅 [ISO 认证证书⁠（在新窗口中打开）](https://trust.openai.com/?a9437b=a9437b&itemUid=5c93ca3e-519a-4260-a50a-a9de78294538&source=click) 和 [SOC 2 审计报告⁠（在新窗口中打开）](https://trust.openai.com/?itemUid=fa950d02-cbb3-4010-b917-7137a7c2a982&source=click)。

**Role-based access controls:** ChatGPT Enterprise and Edu admins can [create custom roles and set group-level permissions⁠(opens in a new window)](https://help.openai.com/en/articles/11750701-rbac) for tools like connectors, projects, GPTs, search, and more. This helps align access with company policies to streamline management as workspaces grow.

**基于角色的访问控制（RBAC）**：ChatGPT 企业版与教育版管理员可 [创建自定义角色并设置分组级权限⁠（在新窗口中打开）](https://help.openai.com/en/articles/11750701-rbac)，适用于连接器、项目、GPTs、搜索等各类功能模块。此举有助于将访问权限策略与企业既有制度对齐，从而在工作区规模持续扩大的过程中，显著提升管理效率。

**Expanded SSO support:** We rolled out [OIDC support⁠(opens in a new window)](https://help.openai.com/en/articles/9534785-configuring-sso-for-chatgpt) for ChatGPT Business, Enterprise, Edu, and API customers, in addition to SAML, giving customers more flexibility to integrate with their existing identity providers.

**增强的单点登录（SSO）支持**：除原有 SAML 协议外，我们还为 ChatGPT 商业版、企业版、教育版及 API 客户推出了 [OIDC 协议支持⁠（在新窗口中打开）](https://help.openai.com/en/articles/9534785-configuring-sso-for-chatgpt)，为客户对接现有身份提供商（IdP）提供更多灵活性。

**IP allowlisting**: Admins can now enable [IP allowlisting⁠(opens in a new window)](https://help.openai.com/en/articles/12111596-ip-allowlisting-for-chatgpt) as an optional security feature for ChatGPT Enterprise and Edu workspaces. When turned on, only requests from the IP addresses you specify will be allowed. Any request from an unapproved IP is automatically blocked, even if the user has valid credentials.

**IP 白名单**：管理员现可为 ChatGPT 企业版与教育版工作区启用 [IP 白名单⁠（在新窗口中打开）](https://help.openai.com/en/articles/12111596-ip-allowlisting-for-chatgpt) 这一可选安全功能。启用后，仅允许来自您指定 IP 地址的请求接入；任何源自未授权 IP 的请求将被自动拦截——即使该用户拥有合法有效的登录凭证。

These updates build on our existing, robust enterprise-grade security, data privacy, and compliance programs. [Learn more⁠](https://openai.com/business-data/).

这些更新建立在我们现有成熟、稳健的企业级安全、数据隐私与合规计划基础之上。[了解更多信息⁠](https://openai.com/business-data/)。
