Nov 20, 2020

![](images/projects-can-now-be-renamed-vercel/img_001.jpg)

For all the Projects you've deployed on Vercel in the past, the platform either automatically selected a name for you based on the name of your Git repository or local directory, or you manually customized it before the Project was created.

Previously, it wasn't possible to change the name of a Project after it was created. As of today, however, you can do it directly from the "General" page in the Project Settings.

When changing the name of your Project, no interruptions in your or your Team's workflow will occur, considering that you're either [deploying from a Git repository](https://vercel.com/docs/git) or you've [linked the Project to a local directory](https://vercel.com/docs/cli#commands/overview/project-linking) using Vercel CLI.

Check out [the documentation](https://vercel.com/docs/platform/projects#project-name) as well.
