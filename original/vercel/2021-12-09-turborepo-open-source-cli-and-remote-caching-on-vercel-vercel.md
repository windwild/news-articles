[Turborepo](https://www.turborepo.org/), a high-performance build system for JavaScript and TypeScript codebases, is joining Vercel. Starting today, the [Turborepo CLI](https://turborepo.org/docs/getting-started) is open source and available for anyone to use.

Turborepo reduces build times by providing:

- **Intelligent caching**

- **Content-aware hashing**

- **Remote caching (beta)**

- **Parallel execution**

- **Task pipelines**


[Get started](https://turborepo.org/docs/getting-started) with Turborepo today.
