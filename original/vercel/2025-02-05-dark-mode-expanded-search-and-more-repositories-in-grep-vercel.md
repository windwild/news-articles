Feb 5, 2025

![](images/dark-mode-expanded-search-and-more-repositories-in-grep-vercel/img_001.jpg)

We've made improvements to [Grep](https://grep.app/), our tool for quick code search.

- You can now search across 1,000,000 public git repositories

- The app has been rebuilt with [Next.js 15](https://nextjs.org/docs), improving performance with [Partial Prerendering](https://nextjs.org/learn/dashboard-app/partial-prerendering)

- Support for dark mode


[Try Grep today](https://grep.app/).
