[Edge Functions](https://vercel.com/features/edge-functions) are now in public beta. Edge API Routes, which use Edge Functions, enable you to create high-performance APIs for use with any frontend framework. These functions use the same standard Web APIs as Edge Middleware, like `Request`, `Response`, and `fetch`.

[Check out the documentation](https://vercel.com/docs/concepts/functions/edge-functions) to get started.
