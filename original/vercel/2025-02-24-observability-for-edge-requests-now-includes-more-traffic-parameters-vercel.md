We’ve expanded the [Edge Request dashboard](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fobservability%2Fedge-requests&title=Edge+Request+Observability) in [Vercel Observability](https://vercel.com/docs/observability) to show additional request data by:

- User agent

- IP address

- JA4

- Referrer

- Hostname


Available on all plans, these insights help you monitor traffic patterns and identify potential threats, which you can address using [Vercel Firewall](https://vercel.com/docs/security/vercel-firewall).

Route-level data is available to [Observability Plus](https://vercel.com/docs/observability#observability-plus) customers.

[View your Edge Request dashboard](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fobservability%2Fedge-requests&title=Edge+Request+Observability) and learn more about [Vercel Observability](https://vercel.com/docs/observability).
