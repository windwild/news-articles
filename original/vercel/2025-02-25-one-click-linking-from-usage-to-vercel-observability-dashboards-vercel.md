![](images/one-click-linking-from-usage-to-vercel-observability-dashboards-vercel/img_001.jpg)![](images/one-click-linking-from-usage-to-vercel-observability-dashboards-vercel/img_002.jpg)

Metrics on the [Usage dashboard](https://vercel.com/docs/pricing/manage-and-optimize-usage) now offer one-click access to corresponding [Vercel Observability](https://vercel.com/docs/observability) dashboards, making it easier to dive deeper into team and project usage.

This new linking is available today for:

- Vercel Functions

- Edge Network

- Image Optimization

- Incremental Static Regeneration

- Builds (when viewing per project)


Try it from [your Usage dashboard](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fusage&title=View+Usage+dashboard) and learn more about [Vercel Observability](https://vercel.com/docs/observability).
