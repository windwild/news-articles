Dec 12, 2025

![](images/referer-now-available-in-runtime-logs-vercel/img_001.jpg)![](images/referer-now-available-in-runtime-logs-vercel/img_002.jpg)

For any request displayed on runtime logs in the Vercel dashboard, you can now view the referer (if any) for that request in the right hand details panel.

This allows you to understand the source of that request and more easily debug issues.

[Try it out](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Flogs) or learn more about [Runtime Logs](https://vercel.com/docs/logs/runtime).
