![](images/improved-web-notifications-are-now-generally-available-vercel/img_001.jpg)![](images/improved-web-notifications-are-now-generally-available-vercel/img_002.jpg)

Web notifications, [previously in a public beta](https://vercel.com/changelog/new-notification-controls-available-in-public-beta), are now generally available. Two improvements we have made for this release include:

- Deployment failure notifications will not land in your web inbox if you have already navigated to the deployment's page.

- You can now opt out of domain configuration notifications. While we do not recommend this action to avoid any application downtime, you are able to set this as a preference.


[Check out the documentation](https://vercel.com/docs/concepts/dashboard-features/notifications) to learn more.
