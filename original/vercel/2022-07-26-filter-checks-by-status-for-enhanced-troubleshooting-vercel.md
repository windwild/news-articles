You can now filter Checks by status to show which failures are causing performance regressions. Install the [Checkly Integration](https://vercel.com/integrations/checkly) to add auto-generated Web Vitals monitoring to your deployments and prevent performance regressions.

To build your own deployment validation and status checks, [view the Checks API documentation](https://vercel.com/docs/integrations/checks-overview#building-your-own-checks).
