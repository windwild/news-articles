Vercel now supports and automatically optimizes your [Hydrogen 2 projects](https://vercel.com/templates/remix/hydrogen-2) as of Vercel CLI `v31.2.3`. When importing a new project, it will detect Hydrogen and configure the right settings for optimal performance — including using [Vercel Edge Functions](https://vercel.com/features/edge-functions) for server-rendering pages.

Deploy the [Hydrogen template](https://vercel.com/templates/remix/hydrogen-2) or run `vercel init hydrogen-2` command in your terminal to get started.
