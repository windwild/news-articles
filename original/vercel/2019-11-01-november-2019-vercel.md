Nov 1, 2019

## Dashboard

- The status of a deployment is now reflected in the tab icon

- External domains are now automatically assigned to your latest production deployment when added to your project

- The warning for adding more than 3 members to a free team is now more clear

- Added a warning for expired team invite links

- Deploying an example now includes a screenshot and link to a live example

- Improved the look and feel of the dashboard and projects pages

- Fixed incorrect DNS indication on Domains page when adding an external domain

- Improved 404 and error pages across the dashboard with detailed information and links to documentation

- Node.js 12 LTS is now available for Serverless Functions ( [read more](https://zeit.co/docs/v2/serverless-functions/supported-languages#defined-node.js-version))

- Refined domain notification emails with clear instructions on how to properly configure your domain
