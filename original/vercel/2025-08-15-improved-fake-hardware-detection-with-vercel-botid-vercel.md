Aug 15, 2025

[Vercel BotID Deep Analysis](https://vercel.com/docs/botid/get-started) now uses an updated detection model that expands fingerprinting coverage for bespoke headless browsers and simulated device hardware.

[BotID](https://vercel.com/blog/introducing-botid) is an invisible CAPTCHA that classifies sophisticated bots without interrupting real users. The new Deep Analysis model enables more accurate identification of stealthy automation frameworks and spoofed hardware profiles in real time.

These updates take effect immediately for BotID Deep Analysis users with no action required, but we recommend upgrading to the latest `botid@1.5.2`.

Get started with [BotID today](https://vercel.com/docs/botid).
