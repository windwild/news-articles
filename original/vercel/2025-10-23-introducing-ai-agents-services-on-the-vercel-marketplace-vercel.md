The Vercel Marketplace now includes a dedicated [**AI Agents & Services**](https://vercel.com/marketplace/category/agents) category, making it easier for developers to integrate AI-powered automation, observability, and infrastructure directly into their projects.

This category introduces native support for agentic integrations with unified authentication, provisioning, and billing across providers, all within the Vercel platform.

**Agents -** Off-the-shelf agents that reason and act on your behalf

- [**CodeRabbit**](https://vercel.com/marketplace/coderabbit) **:** Automated code review and PR feedback

- [**Corridor**](https://vercel.com/marketplace/corridor) **:** Real-time security and threat detection

- [**Sourcery**](https://vercel.com/marketplace/sourcery) **:** Code review and generation assistance


**AI services -** Infrastructure for building and scaling your own agents

- [**Braintrust**](https://vercel.com/marketplace/braintrust) **:** Evaluation and monitoring frameworks

- [**Kernel**](https://vercel.com/marketplace/kernel) **:** Cloud browser infrastructure for agentic workloads

- [**Mixedbread**](https://vercel.com/marketplace/mixedbread) **:** Multimodal AI search and retrieval across documents, code, media, and more

- [**Kubiks**](https://vercel.com/marketplace/kubiks) **:** Multi-step workflow orchestration and remediation

- [**Chatbase:**](https://vercel.com/marketplace/chatbase) Analytics and tuning for conversational agent

- [**Autonoma**](https://vercel.com/marketplace/autonoma-ai) **:** AI-driven testing for your web and mobile apps to build, run, and analyze without writing code

- [**Browser Use**](https://vercel.com/marketplace/browseruse) **:** Natural language browser control for web automation

- [**Descope**](https://vercel.com/marketplace/descope) **:** No-code identity and authentication workflows for users, partners, and AI agents


Explore the [agentic marketplace](https://vercel.com/marketplace/category/agents), read our [blog](https://vercel.com/blog/ai-agents-and-services-on-the-vercel-marketplace), and check out the [documentation](https://vercel.com/docs/agent-integrations).
