Comments on Vercel [Preview Deployments](https://vercel.com/docs/concepts/deployments/preview-deployments) can now be converted to Linear issues directly from the comment. This makes it easy to take action on feedback in the workflows your team is already using.

This feature is available to Pro and Enterprise users as well as Hobby users with public repos.

To get started, check out our [Vercel Linear integration](https://vercel.com/integrations/linear) in the Vercel Integrations Marketplace. You can select the **Add Integration** button from within the Marketplace, then select which Team and project the integration should be scoped to.

[Check out our documentation](https://vercel.com/docs/concepts/deployments/comments/integrations#use-the-vercel-comments-linear-integration) to learn more.
