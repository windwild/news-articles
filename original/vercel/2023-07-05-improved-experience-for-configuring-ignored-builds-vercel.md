![](images/improved-experience-for-configuring-ignored-builds-vercel/img_001.jpg)![](images/improved-experience-for-configuring-ignored-builds-vercel/img_002.jpg)

You no longer have to write your own commands when configuring your project's [Ignored Build Step](https://vercel.com/docs/concepts/projects/overview#ignored-build-step). We've looked at the most commonly used scenarios to create presets for an easier experience:

- Choose a preset to use common configurations of Vercel customers

- Select a Node or Bash script from your repository

- Write an arbitrary Bash script using the "Custom" option


[Check the documentation](https://vercel.com/docs/concepts/projects/overview#ignored-build-step) to learn more about Ignored Build Step.
