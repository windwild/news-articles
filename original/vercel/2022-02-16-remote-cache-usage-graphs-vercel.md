Feb 16, 2022

![](images/remote-cache-usage-graphs-vercel/img_001.jpg)

Remote Caching for Monorepo tools now includes usage graphs for your team, including:

- Total number of artifacts uploaded or downloaded

- Total size of artifacts successfully uploaded or downloaded


Monorepo tools like [Turborepo](https://vercel.com/docs/concepts/git/monorepos#turborepo) can now use Remote Caching on Vercel with zero configuration. Check out [the documentation](https://vercel.com/docs/concepts/limits/usage#artifacts).
