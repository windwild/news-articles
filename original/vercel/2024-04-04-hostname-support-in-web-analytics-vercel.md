![](images/hostname-support-in-web-analytics-vercel/img_001.jpg)![](images/hostname-support-in-web-analytics-vercel/img_002.jpg)

You can now inspect and filter hostnames in Vercel Web Analytics.

- **Domain insights:** Analyze traffic by specific domains. This is beneficial for per-country domains, or for [building multi-tenant applications](https://vercel.com/templates/next.js/platforms-starter-kit).

- **Advanced filtering:** Apply filters based on hostnames to view page views and custom events per domain.


This feature is available to all Web Analytics customers.

Learn more in [our documentation](https://vercel.com/docs/analytics/filtering#using-filters) about filtering.
