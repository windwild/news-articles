Vercel now supports [Git LFS](https://vercel.com/docs/concepts/projects/overview#git-large-file-storage-lfs), enabling storing large files using your git client of choice.

Git LFS is free for all plans and can be toggled on and off in your project settings. File storage may consume additional bandwidth with your git provider that stores files (GitHub, GitLab, Bitbucket, etc.).

[Check out our documentation](https://vercel.com/docs/concepts/projects/overview#git-large-file-storage-lfs) to learn more.
