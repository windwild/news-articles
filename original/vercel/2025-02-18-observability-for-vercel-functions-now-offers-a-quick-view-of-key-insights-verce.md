[Observability](https://vercel.com/docs/observability)'s [Vercel Functions dashboard](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fobservability%2Fvercel-functions&title=Go+to+Observability) now shows quick-view tiles with key metrics, such as:

- Active compute model, like [Fluid compute](https://vercel.com/docs/functions/fluid-compute), which enhances efficiency, minimizes cold starts, and optimizes performance

- Compute saved with Fluid compute enabled

- Average memory usage for your functions

- P75 Time to First Byte (TTFB) for performance monitoring

- Cold start frequency to track optimization impact


These insights are available for all plans.

Learn more about [Observability](https://vercel.com/docs/observability) and [Fluid compute](https://vercel.com/docs/functions/fluid-compute).
