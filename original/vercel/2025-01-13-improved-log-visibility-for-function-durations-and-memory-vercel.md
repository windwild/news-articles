![](images/improved-log-visibility-for-function-durations-and-memory-vercel/img_001.jpg)![](images/improved-log-visibility-for-function-durations-and-memory-vercel/img_002.jpg)

Logs now indicate when Vercel Functions reach (or near) their maximum duration or memory allocation for each request.

Logs also include quick links to configure function maximum duration, CPU & memory, region, and Node.js Version directly from requests.

View [your project's logs](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Flogs&title=Go+to+Logs).
