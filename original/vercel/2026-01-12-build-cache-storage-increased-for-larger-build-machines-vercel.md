Jan 12, 2026

Projects that use [larger build machines](https://vercel.com/docs/builds/managing-builds#larger-build-machines) now have increased limits for build caches.

- Enhanced build machines have a 3 GB limit

- Turbo build machines have a 7 GB limit


These limit increases come at no additional charge.

Learn more about [Vercel's build cache](https://vercel.com/docs/deployments/troubleshoot-a-build#understanding-build-cache).
