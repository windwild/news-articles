May 19, 2025

![](images/hypertune-joins-the-vercel-marketplace-vercel/img_001.jpg)![](images/hypertune-joins-the-vercel-marketplace-vercel/img_002.jpg)

[Hypertune](https://www.hypertune.com/) now offers a native integration with Vercel Marketplace.

You can find it as a Flags & Experimentation provider in the Flags tab.

The Hypertune integration offers:

- Powerful flags, A/B testing, analytics and app configuration

- Full end-to-end type safety with [type-safe client generation](https://docs.hypertune.com/sdk-reference/type-safe-client-generation)

- Personalization with the lowest latency using [Edge Config](https://vercel.com/blog/edge-config-ultra-low-latency-data-at-the-edge-3dltgn2G31EpEubVHc5Hrs)

- A first class [Flags SDK adapter](https://flags-sdk.dev/providers/hypertune)


Install and access on Vercel with one-click setup and unified billing.

Deploy the [Hypertune template](https://vercel.com/templates/Next.js/flags-sdk-hypertune-nextjs) built for Vercel Marketplace today.
