![](images/project-settings-are-now-searchable-vercel/img_001.jpg)![](images/project-settings-are-now-searchable-vercel/img_002.jpg)

You can now search within project settings in the [Vercel Dashboard](https://vercel.com/docs/dashboard-features#dashboard-overview), making it easier to quickly find a specific setting.

To get to [your project settings](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fsettings&title=Open+project+settings):

1. Select a project from your **Team Overview** page

2. Select the **Settings** tab.


Learn more about [project settings](https://vercel.com/docs/projects/project-configuration/project-settings).
