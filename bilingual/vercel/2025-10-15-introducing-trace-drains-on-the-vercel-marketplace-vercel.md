Oct 15, 2025

2025年10月15日

![](images/introducing-trace-drains-on-the-vercel-marketplace-vercel/img_001.jpg)![](images/introducing-trace-drains-on-the-vercel-marketplace-vercel/img_002.jpg)

You can now use [Vercel Drains](https://vercel.com/blog/introducing-vercel-drains) to send traces and logs from your projects to your preferred Marketplace observability providers with native integrations, including [Braintrust](https://vercel.com/marketplace/braintrust), [Dash0](https://vercel.com/marketplace/dash0), [Statsig](https://vercel.com/marketplace/statsig), and [Kubiks](https://vercel.com/marketplace/kubiks) with more providers coming soon.

您现在可以使用 [Vercel Drains](https://vercel.com/blog/introducing-vercel-drains)，通过原生集成，将项目中的追踪（traces）和日志（logs）发送至您首选的 Marketplace 可观测性服务商，包括 [Braintrust](https://vercel.com/marketplace/braintrust)、[Dash0](https://vercel.com/marketplace/dash0)、[Statsig](https://vercel.com/marketplace/statsig) 和 [Kubiks](https://vercel.com/marketplace/kubiks)，更多服务商即将上线。

This integration allows developers to stream traces and evaluation data from Vercel directly into these providers for observability, debugging, and performance monitoring.

该集成使开发者能够将来自 Vercel 的追踪数据与评估数据实时流式传输至上述服务商，用于可观测性分析、调试及性能监控。

The Trace Drain API extends Vercel’s observability surface to the Marketplace ecosystem, allowing providers to:

Trace Drain API 将 Vercel 的可观测性能力延伸至 Marketplace 生态系统，赋能服务商实现以下能力：

- Deliver rich visibility into performance and debugging data  
- 提供对性能与调试数据的深度可视化洞察  

- Integrate natively with logging, and analytics tools through Vercel  
- 通过 Vercel 原生集成日志与分析工具  

- Build tighter feedback loops between deployments and infrastructure insights  
- 在部署与基础设施洞察之间构建更紧密的反馈闭环  

- Offer customers a fully connected experience without manual setup  
- 为客户提供开箱即用、端到端贯通的体验，无需手动配置  

This update gives teams more flexibility to use their preferred observability tools while maintaining a single, unified developer experience inside Vercel.

本次更新赋予团队更大灵活性——可自由选用偏好的可观测性工具，同时在 Vercel 内部持续享有统一、一致的开发者体验。

[Try it out](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fdrains) or [learn more](http://vercel.com/docs/drains#getting-started-with-drains) about this update available to Pro and ENT customers.

[立即试用](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fdrains) 或 [了解详情](http://vercel.com/docs/drains#getting-started-with-drains)，此更新面向 Pro 和 ENT 用户开放。
