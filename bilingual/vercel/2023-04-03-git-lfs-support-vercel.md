![Git LFS 将音频样本、视频、数据集和图形等大文件替换为 Git 内部的纯文本指针，同时将文件内容存储在远程服务器（如 GitHub.com 或 GitHub Enterprise）上。](images/git-lfs-support-vercel/img_001.jpg)![Git LFS 将音频样本、视频、数据集和图形等大文件替换为 Git 内部的纯文本指针，同时将文件内容存储在远程服务器（如 GitHub.com 或 GitHub Enterprise）上。](images/git-lfs-support-vercel/img_002.jpg)

Vercel 现已支持 [Git LFS](https://vercel.com/docs/concepts/projects/overview#git-large-file-storage-lfs)，让您能够使用自己偏好的 Git 客户端存储大型文件。

Git LFS 对所有套餐均免费，并可在项目设置中随时启用或禁用。文件存储可能会额外消耗您所使用的 Git 托管服务商（如 GitHub、GitLab、Bitbucket 等）的带宽。

[查阅我们的文档](https://vercel.com/docs/concepts/projects/overview#git-large-file-storage-lfs) 以了解更多信息。
