Jan 13, 2023

2023 年 1 月 13 日

Swell 是一个构建在 Vercel 平台上的电商平台，它允许任何人借助其无头（headless）、以 API 为先的后端，快速搭建专属的电商网站。对 Swell 而言，Vercel 与 Next.js 同时提供了所需的灵活性与易用性，从而赋能其全球用户打造高性能的线上商店。这一合作带来双重优势：一方面，Vercel 和 Next.js 为 Swell 团队提供了颠覆性的工具与功能；另一方面，它们也确保 Swell 的商家能为其客户构建出加载速度最快、购物体验最佳的网站。

**“我们看到商家正在 Swell 上构建这些极速响应的网站。而这正是无头电商的核心优势之一——你可以采用业界顶尖的技术栈，例如 Vercel 和 Next.js。” — Greg Hoskin（解决方案工程负责人）**

许多商家此前使用的是传统电商系统，他们希望在提升营销能力的同时，不牺牲网站性能。得益于 Vercel 的**按需增量静态再生（ISR）**（[on-demand Incremental Static Regeneration](https://vercel.com/blog/how-hashicorp-developers-iterate-faster-with-isr#what-is-isr)）和混合渲染能力，Swell 成功实现了这一目标——让商家可自由上架无限数量的 SKU，而丝毫不影响网站加载速度。此外，上线流程极为顺畅：一位近期客户借助 Swell 及 [Next.js Commerce 启动套件](https://nextjs.org/commerce)，仅用一个月便完成了整站搭建；相比之下，传统平台通常需要四个月至一年不等。

**“我们的商家能够快速启动并运行一个既高速又对开发者友好的框架；同时，还能在此基础之上深度扩展与高度定制其店铺及整体电商体验。”**

不仅商家为 Vercel 和 Next.js 所展现的多功能性与灵活性而振奋，Swell 团队自身亦深感欣喜。从原有方案迁移至 Vercel 的过程无需任何故障排查，也毫无压力。他们尤其喜爱 Vercel 的控制台（dashboard），因为它让新增平台、管理子域名变得轻而易举。无论是 API 集成、部署流程，还是操作体验与用户界面，其开发者体验均堪称卓越——因为一切皆“开箱即用”。
