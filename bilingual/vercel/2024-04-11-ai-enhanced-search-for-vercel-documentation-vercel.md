2024年4月11日

![](images/ai-enhanced-search-for-vercel-documentation-vercel/img_001.jpg)![](images/ai-enhanced-search-for-vercel-documentation-vercel/img_002.jpg)

You can now get AI-assisted answers to your questions directly from the Vercel docs search:

您现在可直接通过 Vercel 文档搜索功能，获取 AI 辅助回答：

- Use natural language to ask questions about the docs

- 使用自然语言向文档提问

- View recent search queries and continue conversations

- 查看最近的搜索记录，并延续对话

- Easily copy code and markdown output

- 轻松复制代码和 Markdown 格式输出

- Leave feedback to help us improve the quality of responses

- 提交反馈，帮助我们提升回答质量

Start searching with `⌘K` (or `Ctrl+K` on Windows) menu on [vercel.com/docs](https://vercel.com/docs).

请在 [vercel.com/docs](https://vercel.com/docs) 页面中，通过 `⌘K`（Windows 系统为 `Ctrl+K`）快捷键打开搜索菜单，立即开始搜索。
