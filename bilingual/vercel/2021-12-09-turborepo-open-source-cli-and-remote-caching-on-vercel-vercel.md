[Turborepo](https://www.turborepo.org/) 是一个面向 JavaScript 和 TypeScript 代码库的高性能构建系统，现正式加入 Vercel。即日起，[Turborepo CLI](https://turborepo.org/docs/getting-started) 开源发布，供所有人免费使用。

Turborepo 通过以下特性显著缩短构建时间：

- **智能缓存（Intelligent caching）**

- **内容感知哈希（Content-aware hashing）**

- **远程缓存（Remote caching，测试版）**

- **并行执行（Parallel execution）**

- **任务流水线（Task pipelines）**

立即 [开始使用 Turborepo](https://turborepo.org/docs/getting-started)。
