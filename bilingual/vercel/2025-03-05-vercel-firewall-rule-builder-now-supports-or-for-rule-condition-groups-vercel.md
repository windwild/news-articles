2025年3月5日

![](images/vercel-firewall-rule-builder-now-supports-or-for-rule-condition-groups-vercel/img_001.jpg)![](images/vercel-firewall-rule-builder-now-supports-or-for-rule-condition-groups-vercel/img_002.jpg)

The Vercel Firewall now supports using an `OR` operator to link condition groups within a custom WAF rule.

Vercel 防火墙现已支持在自定义 WAF 规则中使用 `OR`（或）运算符来连接条件组。

Previously, customers could only use an `AND` operator to join condition groups. This update now supports `AND` or `OR` , allowing customers to create more complex WAF actions.

此前，用户仅能使用 `AND`（与）运算符连接条件组。本次更新同时支持 `AND` 和 `OR` 运算符，使用户能够构建更复杂的 WAF 动作。

Learn more about the [Vercel Firewall](https://vercel.com/docs/security/vercel-firewall) or navigate to your [Firewall tab](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Ffirewall&title=Firewall+tab) to customize rules.

了解更多关于 [Vercel 防火墙](https://vercel.com/docs/security/vercel-firewall) 的信息，或前往您的 [防火墙标签页](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Ffirewall&title=Firewall+tab) 自定义规则。
