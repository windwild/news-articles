May 7, 2025

2025年5月7日

![](images/faster-builds-now-available-with-compute-upgrades-on-paid-plans-vercel/img_001.jpg)![](images/faster-builds-now-available-with-compute-upgrades-on-paid-plans-vercel/img_002.jpg)

Projects with [on-demand concurrent builds](https://vercel.com/docs/builds/managing-builds#on-demand-concurrent-builds) can now use enhanced build machines to improve build performance.

支持[按需并发构建](https://vercel.com/docs/builds/managing-builds#on-demand-concurrent-builds)的项目，现在可使用增强型构建机器来提升构建性能。

Available on all paid plans, these machines offer double the resources: 8 CPUs, 16 GB memory, and 58 GB disk. This reduces both build time and total build minutes used. Existing customers are already seeing up to 25% faster builds with no changes required.

该增强型构建机器面向所有付费方案开放，资源配置提升一倍：8 核 CPU、16 GB 内存、58 GB 磁盘空间。这不仅缩短了单次构建耗时，也降低了总构建分钟数。现有客户在无需任何配置变更的情况下，已实现最高达 25% 的构建速度提升。

Enhanced builds can be enabled per project and are billed per minute.

增强型构建可按项目单独启用，并按实际使用分钟数计费。

Enterprise customers can run all concurrent builds, including pre-allocated build slots and on-demand, on higher-spec machines.

企业版客户可将全部并发构建（包括预分配构建槽位与按需构建）均运行于更高规格的机器上。

[Enable on-demand enhanced builds](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fsettings%2Fbuild-and-deployment&title=Enable%20enhanced%20builds) and learn more in [our documentation](https://vercel.com/docs/builds/managing-builds#enhanced-build-machines).

[启用按需增强型构建](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fsettings%2Fbuild-and-deployment&title=Enable%20enhanced%20builds)，并在[我们的文档](https://vercel.com/docs/builds/managing-builds#enhanced-build-machines)中了解更多信息。
