Nov 20, 2025

2025 年 11 月 20 日

![](images/improved-analytics-experience-now-available-on-the-vercel-firewall-vercel/img_001.jpg)![](images/improved-analytics-experience-now-available-on-the-vercel-firewall-vercel/img_002.jpg)

We have launched improvements to the Vercel Firewall UI, simplifying your application security monitoring and analysis. Vercel Firewall includes the System firewall and DDoS mitigations, Web Application Firewall, and Bot Management capabilities.

我们已上线 Vercel 防火墙（Vercel Firewall）用户界面的多项改进，使您的应用安全监控与分析工作更加简洁高效。Vercel 防火墙涵盖系统防火墙与 DDoS 缓解、Web 应用防火墙（WAF）以及机器人管理（Bot Management）等功能。

The updated experience surfaces more information on security events and mitigations, and allows for easier event investigations, bringing together all security events analytics in one place.

此次更新显著增强了安全事件与缓解措施的信息呈现，并简化了事件调查流程，将全部安全事件分析能力整合于统一界面中。

The updates include:

更新内容包括：

- **An updated Overview page** for a consolidated view of DDoS attacks, and activity across system rules, custom rules, and IP blocks.

- **更新后的概览页（Overview page）**：集中展示 DDoS 攻击概况，以及系统规则、自定义规则和 IP 封禁等各项活动。

- **A new Traffic page** that allows you to drill down into top sources of traffic (IPs, request paths, JA4 digests, ASN, user agents) and filter by actions (allowed, logged, denied, challenged, rate limited)

- **全新流量页（Traffic page）**：支持深入分析主要流量来源（包括 IP 地址、请求路径、JA4 摘要、ASN 和用户代理），并可按操作类型（放行、记录、拒绝、质询、限速）进行筛选。

- **Simplified UX for writing custom rules or queries** so that you can take actions or do analysis without friction

- **简化的自定义规则与查询编写体验（UX）**：让您能够更顺畅地执行操作或开展分析，大幅降低使用门槛。

Learn more about the [Firewall](https://vercel.com/docs/vercel-firewall) or visit the [Firewall tab](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Ffirewall&title=Firewall+tab) on your project to see the updates.

如需了解更多信息，请参阅 [防火墙文档](https://vercel.com/docs/vercel-firewall)，或前往您项目的 [防火墙标签页（Firewall tab）](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Ffirewall&title=Firewall+tab) 查看本次更新。
