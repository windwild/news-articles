2021 年 7 月 28 日

![](images/get-started-faster-with-the-new-project-creation-flow-vercel/img_001.jpg)

现在，在控制台中创建新项目将引导您进入一个全新、更简化的流程：

- 新项目将**默认使用您当前选定的个人账户或团队**，而不再每次都需要您手动选择。

- 所有操作现在都集中在**单个视图中完成**，无需再反复在四个不同页面之间跳转。

这些改进让导入 Git 仓库或从模板全新开始变得更加轻松。立即点击控制台上的 [New Project（新建项目）](https://vercel.com/new) 按钮体验吧！

如果您希望将我们的全新项目创建流程集成到您自己的应用程序或 Git 仓库中，也欢迎查阅 [相关文档](https://vercel.com/docs/more/deploy-button)。
