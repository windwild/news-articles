# OpenAI Codex and Figma launch seamless code-to-design experience

# OpenAI Codex 与 Figma 推出无缝的“代码到设计”体验

Key takeaways:

核心要点：

- New Codex to Figma integration helps users move seamlessly between code and the design canvas to iterate and ship products faster.

- 全新的 Codex 到 Figma 集成，帮助用户在代码与设计画布之间无缝切换，从而加速产品迭代与交付。

- The Figma MCP Server connects Codex directly to its design platform and tools like Figma Make and FigJam.

- Figma MCP Server 将 Codex 直接接入其设计平台，以及 Figma Make、FigJam 等工具。

- The new integration expands the partnership between the two companies, which includes the Figma app in ChatGPT, and bringing the latest OpenAI models to Figma’s platform.

- 此次新集成进一步拓展了两家公司的合作关系，此前合作已包括 ChatGPT 中的 Figma 应用，以及将最新的 OpenAI 模型引入 Figma 平台。

OpenAI and Figma are deepening their partnership with a new code-to-design integration that connects Figma directly to Codex. Product builders can easily generate Figma designs from Codex and implement designs from Figma files back into code.

OpenAI 与 Figma 正通过一项全新的“代码到设计”集成，进一步深化双方合作——该集成实现了 Figma 与 Codex 的直接连接。产品构建者可轻松地从 Codex 生成 Figma 设计，也可将 Figma 文件中的设计反向实现为代码。

Figma is a design and product development platform that lets teams create, prototype, and iterate on digital products together in real time. Using MCP, an open-source standard that enables AI agents to interface with external data sources, applications, and tools, the Codex to Figma integration supports a faster and smoother way to move from implementation in code to exploration in the design canvas.

Figma 是一款设计与产品开发平台，支持团队实时协同创建、制作原型并迭代数字产品。本集成基于 MCP（Model Context Protocol）——一种开源标准，使 AI 智能体能够对接外部数据源、应用程序及工具。借助 MCP，Codex 与 Figma 的集成提供了一种更快速、更流畅的工作流：从代码实现，自然过渡到设计画布上的探索与验证。

In practice, teams can bring details from [Figma Design⁠(opens in a new window)](https://www.figma.com/design/), [Figma Make⁠(opens in a new window)](https://www.figma.com/make/), or [FigJam⁠(opens in a new window)](https://www.figma.com/figjam/) into Codex to implement them in code. With the new integration, users can now also turn UI from code into editable Figma designs so they can explore new ideas and iterations before bringing the changes back to code.

在实际应用中，团队可将来自 [Figma Design⁠（在新窗口中打开）](https://www.figma.com/design/)、[Figma Make⁠（在新窗口中打开）](https://www.figma.com/make/) 或 [FigJam⁠（在新窗口中打开）](https://www.figma.com/figjam/) 的设计细节导入 Codex，并将其转化为可执行代码。借助此次新集成，用户现在还可将代码中的 UI 反向转换为可编辑的 Figma 设计，从而在将修改同步回代码前，自由探索新构想与多种迭代方案。

The result is a new roundtrip workflow where Figma and OpenAI users can start anywhere, and move forward with speed and attention to craft. Whether a product idea starts from a prompt, in code, or a design, the Figma MCP Server helps connect the best ideas without losing context along the way.

最终成果是一种全新的双向闭环工作流：Figma 与 OpenAI 用户可从任意环节（提示词、代码或设计）出发，以高速度与高精度推进创作。无论产品构想始于一条提示词、一段代码，还是一个设计稿，Figma MCP Server 均能在全程保持上下文连贯，高效串联最优创意。

“As the barriers for building software go down, the amount of software created will increase exponentially. It’s no longer about whether you _can_ build, but _what_ you build and how it stands out,” said Loredana Crisan, Figma’s Chief Design Officer.  
“随着构建软件的门槛不断降低，软件的产出量将呈指数级增长。关键已不再是你‘能否’构建，而在于你‘构建什么’，以及你的成果如何脱颖而出。”Figma 首席设计官洛雷丹娜·克里斯安（Loredana Crisan）表示。

“With this integration, teams can build on their best ideas—not just their first idea—by combining the best of code with the creativity, collaboration, and craft that comes with Figma’s infinite canvas.”  
“借助此次集成，团队得以基于其最优质的想法——而非仅限于最初的想法——开展构建工作：将代码的强大能力，与 Figma 无限画布所承载的创造力、协作力和设计匠心融为一体。”

“This integration makes Codex powerful for a much broader range of builders and businesses because it doesn’t assume you’re ‘a designer’ or ‘an engineer’ first,” said Alexander Embiricos, Codex product lead.  
“此次集成让 Codex 能为更广泛类型的构建者与企业释放强大价值，因为它并不预设用户首先是一位‘设计师’或一位‘工程师’。”Codex 产品负责人亚历山大·恩比里科斯（Alexander Embiricos）表示。

“Engineers can iterate visually without leaving their flow, and designers can work closer to real implementation without becoming full-time coders. The boundary between roles starts to soften because the system helps translate between intent and reality continuously.”  
“工程师无需脱离原有工作流，即可实现可视化迭代；设计师则能在不成为全职编码人员的前提下，更贴近真实实现进行工作。角色之间的边界由此开始消融——因为该系统持续在‘意图’与‘现实’之间架起翻译桥梁。”

The new Codex to Figma integration builds on a strong partnership foundation between both companies. Figma uses OpenAI models in tools across its platform and was one of the first partners to launch a ChatGPT app in 2025. It has also launched ChatGPT Enterprise across its organization to enable AI fluency for its workforce.  
本次全新的 Codex 与 Figma 集成，建立在双方坚实的合作基础之上。Figma 已在其平台各项工具中广泛应用 OpenAI 模型，并于 2025 年成为首批推出 ChatGPT 应用的合作伙伴之一。此外，Figma 已在全公司范围内部署 ChatGPT Enterprise，以全面提升员工的 AI 应用能力。

**Building on Codex momentum**  
**乘势而上：延续 Codex 的发展势头**

Codex first launched as a CLI in January 2025. Codex brings together the most capable agentic coding models with an intuitive product experience that meet builders and developers where they are—whether via the terminal through the CLI, as an IDE extension, on the web, or most recently, the [Codex desktop app](https://openai.com/index/introducing-the-codex-app/) for macOS.  
Codex 于 2025 年 1 月首次以命令行工具（CLI）形式发布。Codex 将当前最具能力的智能体编程模型，与直观易用的产品体验相结合，真正实现了“以构建者为中心”——无论用户习惯通过终端使用 CLI、以 IDE 插件形式调用、在网页端操作，还是最新推出的 [Codex macOS 桌面应用](https://openai.com/index/introducing-the-codex-app/)，Codex 均能无缝适配。

The Codex app enables multitasking with agents and automating background workflows through a seamless experience. Connecting Figma to Codex combines the ease of building with code-first workflows with the craft of designing on a collaborative canvas—unlocking a powerful new way for people to build everything from prototypes to production applications.  
Codex 桌面应用支持用户与多个智能体协同多任务处理，并通过流畅体验自动化后台工作流。将 Figma 与 Codex 相连，既保留了“以代码为先”的构建便捷性，又融合了在协作式画布上精雕细琢的设计工艺——从而开启一种全新而强大的构建范式，助力人们从原型设计一路高效迈向生产级应用开发。

Builders globally are increasingly turning to Codex as a collaborator, from idea through execution. Over 1 million people use Codex weekly, and Codex usage has increased more than 400% since the start of the year. Everyone from product designers, to data scientists, engineers, researchers, and product builders are using Codex to close the gap from inspiration to application.  
全球构建者正日益将 Codex 视为从创意萌芽到落地执行全过程中的得力协作者。目前，每周有超百万人使用 Codex；自今年年初以来，Codex 使用量增幅已逾 400%。无论是产品经理、数据科学家、工程师、研究人员，还是各类产品构建者，都在借助 Codex 缩短从灵感到应用的转化路径。

Companies like Cisco, NVIDIA, Ramp, and Datadog, are among the global enterprises that have adopted Codex for their workforce, along with startups including Harvey and Sierra.  
思科（Cisco）、英伟达（NVIDIA）、Ramp 和 Datadog 等全球知名企业，以及 Harvey、Sierra 等初创公司，均已将 Codex 引入其团队工作流。

To get started, install the [Figma MCP server⁠(opens in a new window)](https://help.figma.com/hc/en-us/articles/32132100833559) directly in the Codex desktop application.  
立即开始：请直接在 Codex 桌面应用中安装 [Figma MCP 服务器⁠（在新窗口中打开）](https://help.figma.com/hc/en-us/articles/32132100833559)。
