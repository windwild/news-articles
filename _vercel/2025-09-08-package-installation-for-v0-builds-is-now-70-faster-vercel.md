Sep 8, 2025

2025 年 9 月 8 日

Average npm install time for v0 builds dropped from 5s to 1.5s by optimizing how dependencies are resolved and cached during build execution.

通过优化构建执行过程中依赖项的解析与缓存机制，v0 构建的平均 `npm install` 时间已从 5 秒缩短至 1.5 秒。

This is in addition to a recent improvement to [time-to-start for v0 builds](https://vercel.com/changelog/faster-time-to-start-for-v0-builds), with more improvements in progress to further reduce installation and overall build time.

此项优化继近期发布的 [v0 构建启动时间（time-to-start）提升](https://vercel.com/changelog/faster-time-to-start-for-v0-builds) 之后推出；目前还有更多优化正在推进中，旨在进一步缩短安装耗时及整体构建时间。

Deploy today on [v0.app](https://v0.app/).

立即在 [v0.app](https://v0.app/) 上部署！
