`@vercel/functions` 现已包含以下新工具函数：

- `geolocation`：返回传入请求的地理位置信息

- `ipAddress`：返回传入请求的 IP 地址

- `getEnv`：返回 Vercel 提供的[系统环境变量](https://vercel.com/docs/projects/environment-variables/system-environment-variables#system-environment-variables)

立即安装最新版包，即可使用这些方法：

```bash
pnpm i @vercel/functions​
```

在[文档](https://vercel.com/docs/functions/vercel-functions-package)中了解更多信息。
