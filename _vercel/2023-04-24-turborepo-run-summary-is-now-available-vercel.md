[Turborepo Run Summary](https://turbo.build/blog/turbo-1-9-0#view-and-compare-task-runs)

[Turborepo 运行摘要](https://turbo.build/blog/turbo-1-9-0#view-and-compare-task-runs) 帮助您直观呈现并调试 Turborepo 的各项任务。

您可以轻松查看部署过程中执行的所有 [任务](https://turbo.build/repo/docs/core-concepts/caching)，包括每项任务的执行耗时、缓存命中状态，以及 `turbo` 用于生成 [缓存键（cache key）](https://turbo.build/repo/docs/core-concepts/caching#hashing) 的数据快照。您还可便捷地对比不同部署结果，快速定位缓存未命中的根本原因，从而消除构建缓慢的问题。

Turborepo 运行摘要现已面向所有用户开放，适用于使用 [`turbo@1.9.0`](https://turbo.build/blog/turbo-1-9-0) 或更高版本的全部订阅计划。

[查阅我们的文档](https://vercel.com/docs/concepts/monorepos/turborepo#debug-turborepo-cache-misses) 以了解更多信息。
