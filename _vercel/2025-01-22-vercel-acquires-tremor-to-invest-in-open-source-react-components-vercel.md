Jan 22, 2025

2025 年 1 月 22 日

Announcing the acquisition of Tremor, bringing elegant dashboard and chart UI components to all developers.

宣布收购 Tremor，为所有开发者带来优雅的仪表板与图表 UI 组件。

[Tremor](https://tremor.so/) is an open source library built on top of React, Tailwind CSS, and Radix. It consists of 35 unique components and 300 blocks that can be copy-pasted to build visually rich and interactive dashboards. The Tremor community has seen impressive growth with over 16,000 stars, 300,000 monthly downloads, and 5,500,000 installs to date.

[Tremor](https://tremor.so/) 是一个基于 React、Tailwind CSS 和 Radix 构建的开源库。它包含 35 个独特组件和 300 个可直接复制粘贴使用的模块（blocks），助力开发者快速构建视觉丰富、交互性强的仪表板。截至目前，Tremor 社区已取得显著增长：GitHub 星标数超 16,000，月下载量达 300,000 次，总安装量达 5,500,000 次。

Today, Tremor and its cofounders [Severin Landolt](https://x.com/severinlandolt) and [Christopher Kindl](https://x.com/kindlaar) are joining Vercel’s Design Engineering team where they'll be working on UI components for the Vercel Dashboard, v0, and more.

今日，Tremor 及其联合创始人 [Severin Landolt](https://x.com/severinlandolt) 和 [Christopher Kindl](https://x.com/kindlaar) 正式加入 Vercel 设计工程团队，将共同致力于 Vercel 仪表板（Vercel Dashboard）、v0 等产品的 UI 组件开发。

This acquisition strengthens our commitment to open source and providing developers with the best tools for building exceptional user interfaces.

此次收购进一步强化了我们对开源的承诺，以及为开发者提供构建卓越用户界面所需最佳工具的决心。

## **Building better dashboards with components**

## **借助组件构建更优仪表板**

Unlike visual dashboard builders, Tremor provides developers with a toolkit of modular, accessible UI elements, offering greater flexibility and control when creating dashboards.

不同于可视化仪表板构建工具，Tremor 为开发者提供了一套模块化、无障碍友好的 UI 元素工具包，使仪表板开发具备更高灵活性与更强控制力。

Vercel’s acquisition of Tremor will provide developers with more tools to build UIs:

Vercel 收购 Tremor 后，将为开发者提供更多构建 UI 的工具：

- **Open source Tremor Blocks:** Along with Tremor's [npm package](https://npm.tremor.so/) and [copy-paste components](https://tremor.so/), Tremor's Pro product— [Tremor Blocks](https://blocks.tremor.so/)—will be free and open source for all users under an MIT license

- **开源 Tremor Blocks**：除 Tremor 现有的 [npm 包](https://npm.tremor.so/) 和 [可复制粘贴组件](https://tremor.so/) 外，其高级产品 —— [Tremor Blocks](https://blocks.tremor.so/) 将面向所有用户免费开放，并以 MIT 许可证形式完全开源。

- **Vercel Dashboard:** Severin and Christopher will contribute their expertise from Tremor to enhance the Vercel Dashboard

- **Vercel 控制台：** Severin 和 Christopher 将把他们在 Tremor 积累的专业知识贡献出来，以增强 Vercel 控制台功能。

- **v0**: Tremor's component library will enhance v0—our [generative UI system](https://v0.dev/)—enabling the creation of more complex and data-rich interfaces from simple text prompts

- **v0：** Tremor 的组件库将赋能 v0——我们的 [生成式 UI 系统](https://v0.dev/)——使开发者仅凭简短文本提示，即可构建更复杂、数据更丰富的用户界面。

## **A commitment to open source**

## **对开源的坚定承诺**

The future of building dashboards is open source, component-driven, and integrated with generative UI. This acquisition supports Vercel's open source commitment and delivers improved tools to streamline UI development for developers.

构建仪表盘的未来在于开源、基于组件，并与生成式 UI 深度融合。此次收购进一步强化了 Vercel 对开源的承诺，并为开发者提供更高效的工具，显著简化 UI 开发流程。
