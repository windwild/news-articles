![](images/edge-function-metrics-now-available-in-monitoring-vercel/img_001.jpg)![](images/edge-function-metrics-now-available-in-monitoring-vercel/img_002.jpg)

监控功能现已新增三项 Edge Function 指标，助您全面掌握 Edge Function 的调用活动与性能表现：

- [Edge Function 调用次数](https://vercel.com/docs/pricing/edge-functions#optimizing-function-invocations) **：** 统计您的 Edge Function 被调用的总次数，包括成功调用与出错调用  

- [Edge Function 执行单元数](https://vercel.com/docs/pricing/edge-functions#managing-execution-units) **：** 衡量您的 Edge Function 所消耗的 CPU 时间，以 50 毫秒为单位进行累加计算  

- [快速源站传输](https://vercel.com/docs/pricing/networking#fast-origin-transfer)（入站与出站） **：** 追踪您的源服务器与 Vercel 之间双向的数据传输速率  


以上指标面向所有已启用 [Observability Plus](https://vercel.com/docs/observability#enabling-observability-plus) 及 [Monitoring](https://vercel.com/docs/observability/monitoring) 功能的客户开放。

监控（Monitoring）功能[近期已正式纳入 Observability Plus 套餐](https://vercel.com/changelog/monitoring-pricing-reduced-up-to-87)。
