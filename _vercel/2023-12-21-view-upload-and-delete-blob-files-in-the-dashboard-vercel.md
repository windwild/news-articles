You can now manage your Vercel Blob files from the dashboard using the new file browser:

您现在可以通过仪表板中的全新文件浏览器来管理您的 Vercel Blob 文件：

- View individual files or folders

- 查看单个文件或文件夹

- Upload new files, including support for drag-and-drop and bulk uploads

- 上传新文件，支持拖放上传和批量上传

- Delete individual files or in bulk

- 删除单个文件或批量删除文件


[Try it out](https://vercel.com/dashboard/stores) or [learn more about Vercel Blob](https://vercel.com/docs/storage/vercel-blob).

[立即试用](https://vercel.com/dashboard/stores) 或 [了解有关 Vercel Blob 的更多信息](https://vercel.com/docs/storage/vercel-blob)。
