![](images/hypertune-integration-available-for-low-latency-experimentation-vercel/img_001.jpg)![](images/hypertune-integration-available-for-low-latency-experimentation-vercel/img_002.jpg)

You can now use the [Hypertune](https://www.hypertune.com/) integration to initialize the Hypertune SDK from [Vercel Edge Config](https://vercel.com/docs/storage/edge-config) with zero latency. This allows you to access your feature flags and run A/B tests with no performance impact to your applications.

您现在可以使用 [Hypertune](https://www.hypertune.com/) 集成，从 [Vercel Edge Config](https://vercel.com/docs/storage/edge-config) 零延迟地初始化 Hypertune SDK。这使您能够访问功能开关（feature flags）并运行 A/B 测试，且对应用性能毫无影响。

This integration is available for users on all plans.

该集成面向所有订阅计划的用户开放。

[Check out the integration](https://vercel.com/integrations/hypertune) to get started.

[立即查看该集成](https://vercel.com/integrations/hypertune)，开始使用。
