The New Relic integration now supports OpenTelemetry traces using Vercel’s new OpenTelemetry Collector. The collector allows users to send traces from Serverless Functions in just a few clicks.

New Relic 集成现已支持通过 Vercel 全新推出的 OpenTelemetry Collector 接收 OpenTelemetry 追踪数据（traces）。借助该 Collector，用户仅需几次点击，即可将无服务器函数（Serverless Functions）的追踪数据发送至 New Relic。

The New Relic integration now includes:

New Relic 集成现已包含以下功能：

1. Vercel Log Drains

1. Vercel 日志导出（Log Drains）

2. OpenTelemetry traces (for Serverless Functions)

2. OpenTelemetry 追踪数据（适用于无服务器函数）

3. A pre-configured dashboard to analyze traces

3. 一个预配置的仪表板，用于分析追踪数据

[Install the integration](https://vercel.com/integrations/newrelic) today or [learn more about](https://vercel.com/docs/integrations/otel-overview/quickstart) how to use OpenTelemetry on Vercel.

[立即安装该集成](https://vercel.com/integrations/newrelic)，或[进一步了解](https://vercel.com/docs/integrations/otel-overview/quickstart)如何在 Vercel 上使用 OpenTelemetry。
