Aug 11, 2025

2025年8月11日

![](images/add-context-when-using-instant-rollback-vercel/img_001.jpg)![](images/add-context-when-using-instant-rollback-vercel/img_002.jpg)

You can now include a reason when performing an Instant Rollback.

执行“即时回滚（Instant Rollback）”时，您现在可以添加回滚原因。

This message is visible to your team in the project overview and can include links or notes explaining the rollback. You can also update it at any time.

该消息将在项目概览页面中对您的团队可见，其中可包含解释回滚原因的链接或备注；您还可随时更新此信息。

Learn more about [Instant Rollback](https://vercel.com/docs/instant-rollback).

了解更多关于[即时回滚（Instant Rollback）](https://vercel.com/docs/instant-rollback)的信息。
