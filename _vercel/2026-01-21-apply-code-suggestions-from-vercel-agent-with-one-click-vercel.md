Jan 21, 2026

2026 年 1 月 21 日

You can now apply suggested code fixes from the Vercel Agent directly in the Vercel Dashboard.

您现在可以直接在 Vercel 控制台中应用 Vercel Agent 提出的代码修复建议。

When the Vercel Agent reviews your pull request, suggestions include a **View suggestion** button that lets you commit the fix to your PR branch, including changes that touch multiple files.

当 Vercel Agent 审查您的 Pull Request 时，每条建议均包含一个 **View suggestion**（查看建议）按钮，点击后即可将该修复提交至您的 PR 分支——即使该修复涉及多个文件的更改。

![Vercel Agent - Review suggestions on dashboard](images/apply-code-suggestions-from-vercel-agent-with-one-click-vercel/img_001.jpg)![Vercel Agent - Review suggestions on dashboard](images/apply-code-suggestions-from-vercel-agent-with-one-click-vercel/img_002.jpg)

Vercel Agent - Review suggestions on dashboard  
Vercel Agent —— 在控制台中审阅建议

Suggestions open in the dashboard, where you can accept them in bulk or apply them one by one.

这些建议将在控制台中展开，您可选择批量接受，也可逐条应用。

![Vercel Agent - Reviewing and applying suggestions in bulk](images/apply-code-suggestions-from-vercel-agent-with-one-click-vercel/img_003.jpg)![Vercel Agent - Reviewing and applying suggestions in bulk](images/apply-code-suggestions-from-vercel-agent-with-one-click-vercel/img_004.jpg)

Vercel Agent - Reviewing and applying suggestions in bulk  
Vercel Agent —— 批量审阅并应用建议

After you apply a suggestion, the review thread is automatically resolved. You can also track multiple concurrent Vercel Agent jobs from the Tasks page.

应用某条建议后，对应的评审讨论线程将自动标记为已解决。您还可通过“Tasks”（任务）页面同时跟踪多个正在运行的 Vercel Agent 任务。

![Vercel Agent - Tracking concurrent agent jobs](images/apply-code-suggestions-from-vercel-agent-with-one-click-vercel/img_005.jpg)![Vercel Agent - Tracking concurrent agent jobs](images/apply-code-suggestions-from-vercel-agent-with-one-click-vercel/img_006.jpg)

Vercel Agent - Tracking concurrent agent jobs

Vercel Agent — 跟踪并发的 Agent 任务

Get started with Vercel Agent code review in the [Agent dashboard](https://vercel.com/~/agent), or learn more in the [documentation](https://vercel.com/docs/agent).

请在 [Agent 控制台](https://vercel.com/~/agent) 中开始使用 Vercel Agent 进行代码审查，或参阅 [文档](https://vercel.com/docs/agent) 了解更多信息。
