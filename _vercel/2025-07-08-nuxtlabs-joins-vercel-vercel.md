Jul 8, 2025

2025年7月8日

[NuxtLabs](https://nuxtlabs.com/)（Nitro 与 Nuxt 的缔造者与守护者）正式加入 Vercel。

[Nuxt](https://nuxt.com/) 是一款全栈 Web 框架，深受数十万开发者的信赖，并被全球诸多顶尖企业采用，每周下载量逾 100 万次。

[Nitro](https://nitro.build/) 是一款面向跨框架、跨平台与多样化工作负载的统一服务端运行时。它为 API、AI 智能体等各类应用提供底层支撑。在许多方面，Nitro 对 Web 框架所起的作用，正如开放容器倡议（Open Container Initiative）之于容器技术：确立了一套清晰、可移植的标准。

NuxtLabs 由 Nuxt 创始人 Sébastien Chopin 领导，长期资助并雇佣了 Nuxt 与 Nitro 核心团队的多位关键成员，包括 Daniel Roe、Pooya Parsa 和 Anthony Fu。他们的工作塑造了现代 Web 开发中广受喜爱的开源生态体系。

## 共同坚守开源承诺

Vercel 致力于推动开放 Web 的持续演进。无论是 Next.js 和 SvelteKit 等框架、Turborepo 和 AI SDK 等工具，还是 shadcn 等独立开发者社区，我们都坚信：最卓越的软件，诞生于开放之中。

**Nuxt 与 Nitro 将继续保持为独立、开源的项目，采用 MIT 许可证，维持公开的路线图与开放的治理机制。** Nitro 将继续以开放、中立、无厂商锁定（vendor-neutral）的方式，服务于所有框架与供应商。社区的核心地位亦将始终不变。

我们深感自豪的是，Vercel 对开源项目的扶持不仅限于基础设施支持，更体现为长期资金投入与共同愿景的深度协同。Vercel 的开源 SDK 战略旨在全面提升 Web 开发标准：与自身基础设施高度内聚（high cohesion），通过设计实现松耦合（loose coupling），并默认全面开放（open by default）。

We want every framework, and every approach to building for the web, to feel fully supported.

我们希望每一个框架、每一种面向 Web 的构建方式，都能获得充分的支持。

## What’s next

## 接下来是什么？

This is a significant milestone for NuxtLabs, Vercel, and the future of the open web. We’re not just backing great software, we’re backing the people behind it.

这是 NuxtLabs、Vercel 以及开放 Web 未来的一个重要里程碑。我们支持的不仅是优秀的软件，更是其背后的开发者。

Over the next few months Nuxt Studio MDC, Nuxt UI Pro, and NuxtHub Admin will all become free and open source.

在接下来的几个月中，Nuxt Studio MDC、Nuxt UI Pro 和 NuxtHub Admin 将全部免费并开源。

We’re honored to welcome the Nuxt team to Vercel to keep shipping for developers everywhere.

我们非常荣幸地欢迎 Nuxt 团队加入 Vercel，持续为全球开发者交付卓越体验。
