You can now filter deployments in the dashboard by author, using username, email, or Git username. Filters persist in the URL, making it easy to share filtered views with your team.

现在，您可在控制台中按作者（支持用户名、邮箱或 Git 用户名）筛选部署。筛选条件会保留在 URL 中，方便您与团队成员共享已筛选的视图。

See now in your [project's deployments](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fdeployments&title=Go+to+Deployments).

立即前往您的 [项目部署页面](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fdeployments&title=Go+to+Deployments) 查看。
