[Edge Functions](https://vercel.com/features/edge-functions) are now in public beta.

[Edge Functions](https://vercel.com/features/edge-functions) 现已开放公开测试。

Edge API Routes, which use Edge Functions, enable you to create high-performance APIs for use with any frontend framework. These functions use the same standard Web APIs as Edge Middleware, like `Request`, `Response`, and `fetch`.

Edge API 路由基于 Edge Functions 构建，使您能够为任意前端框架创建高性能 API。这些函数采用与 Edge 中间件相同的标准化 Web API，例如 `Request`、`Response` 和 `fetch`。

[Check out the documentation](https://vercel.com/docs/concepts/functions/edge-functions) to get started.

[查阅文档](https://vercel.com/docs/concepts/functions/edge-functions) 开始使用。
