![](images/html-element-attribution-in-speed-insights-vercel/img_001.jpg)![](images/html-element-attribution-in-speed-insights-vercel/img_002.jpg)

Speed Insights 现在会显示导致评分偏低的具体 HTML 元素，帮助您快速定位网站中的性能问题。当前支持的指标包括：

- **交互到下一次绘制**（Interaction to Next Paint，INP）

- **累积布局偏移**（Cumulative Layout Shift，CLS）

- **最大内容绘制**（Largest Contentful Paint，LCP）

- **首次输入延迟**（First Input Delay，FID）

该功能面向所有使用 Speed Insights 的客户开放。
