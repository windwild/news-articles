2025 年 12 月 22 日

![](images/function-start-type-now-available-in-runtime-logs-vercel/img_001.jpg)![](images/function-start-type-now-available-in-runtime-logs-vercel/img_002.jpg)

对于任何涉及 Vercel Function 调用的请求，您现在可在 Runtime Logs（运行时日志）右侧的详情面板中查看函数启动类型。

函数调用的启动类型可分为以下三种：`Hot`（热启动）、`Hot (prewarmed)`（预热热启动）或 `Cold`（冷启动）。当函数以 `Cold` 方式启动时，我们还会同时显示冷启动耗时，例如：`Cold (280ms)`。

or learn more about [Runtime Logs](https://vercel.com/docs/logs/runtime).  
或深入了解 [Runtime Logs（运行时日志）](https://vercel.com/docs/logs/runtime)。
