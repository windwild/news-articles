Nov 25, 2021

2021 年 11 月 25 日

![](images/remix-projects-can-now-be-deployed-with-zero-configuration-vercel/img_001.jpg)

Vercel 现在可自动优化您的 [Remix](https://remix.run/) 项目。在导入新项目时，Vercel 将自动识别 Remix，并为您配置合适的设置——包括为 JavaScript 和 CSS 资源自动添加 `immutable` HTTP 缓存响应头。

立即开始：部署 [Remix 模板](https://vercel.com/new/clone?s=https%3A%2F%2Fgithub.com%2Fvercel%2Fvercel%2Ftree%2Fmain%2Fexamples%2Fremix&template=remix&id=67753070&b=main&from=templates)，或运行 `npx create-remix@latest`，在提示中选择 Vercel，然后执行 `npx vercel` 进行部署。

您也可以查阅 [相关文档](https://vercel.com/docs/build-step#build-&-development-settings)。
