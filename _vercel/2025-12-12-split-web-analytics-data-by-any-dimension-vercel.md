Web Analytics now allows you to split data across any dimension.

Web Analytics 现在支持按任意维度拆分数据。

You can now break down your Web Analytics data across any dimension, not just Flags and Flag Values. This update expands support to 11 dimensions, which are:

您现在可以按任意维度（而不仅限于 Flags 和 Flag Values）对 Web Analytics 数据进行细分。本次更新将支持的维度扩展至 11 个，包括：

- Paths  
- Routes  
- Host names  
- Countries  
- Devices  
- OS  

- 路径（Paths）  
- 路由（Routes）  
- 主机名（Host names）  
- 国家/地区（Countries）  
- 设备类型（Devices）  
- 操作系统（OS）

- Referrers  
- 来源网站（引荐来源）

- Flags  
- 标志位（功能开关）

- Flag values  
- 标志位取值

- Events names  
- 事件名称

- Event properties  
- 事件属性

With dimension splits and filters, you can dig deeper into user activity and better understand how different segments are using your application.  

借助维度拆分与筛选功能，您可以更深入地分析用户行为，并更清晰地了解不同用户群体如何使用您的应用。

This feature is available to all Vercel users with the Web Analytics package installed. [Web Analytics Plus](https://vercel.com/docs/analytics/limits-and-pricing#pro-with-web-analytics-plus) subscribers also gain the enhanced capability of splitting data by UTM parameters.  

该功能面向所有已安装 Web Analytics 套件的 Vercel 用户开放。[Web Analytics Plus](https://vercel.com/docs/analytics/limits-and-pricing#pro-with-web-analytics-plus) 订阅用户还可获得增强能力——按 UTM 参数对数据进行拆分。
