Jun 6, 2025

2025 年 6 月 6 日

We've updated our build logic to ensure Git commits authored by [Claude Code](https://github.com/claude) or [Cursor Agent](https://github.com/cursoragent) can trigger deployments on Vercel. A team seat is not required.

我们已更新构建逻辑，确保由 [Claude Code](https://github.com/claude) 或 [Cursor Agent](https://github.com/cursoragent) 提交的 Git 提交会触发 Vercel 上的部署。无需团队席位。

If your agent encounters any issues building on Vercel, please contact us.

若您使用的智能体在 Vercel 上构建时遇到任何问题，请随时联系我们。
