Jan 8, 2021

2021年1月8日

![](images/git-repositories-can-now-be-searched-for-and-imported-easily-vercel/img_001.jpg)

过去，将 Git 仓库导入 Vercel 需要先前往您选择的 Git 托管平台（如 GitHub、GitLab 等），在地址栏中复制该仓库的 URL，再将其粘贴到 Vercel 的项目创建流程中，最后按步骤完成后续操作。

然而，得益于 Vercel 最近的一次更新，[项目创建流程](https://vercel.com/new) 现在会直接呈现一份推荐导入的 Git 仓库列表，并支持对特定仓库进行搜索；此外，每个仓库旁还标注了其所使用的框架图标。

与此同时，“克隆模板”（Cloning a Template）的操作也大幅简化——模板现在与推荐的 Git 仓库一同展示在同一页面上。

也欢迎查阅 [相关文档](https://vercel.com/docs/git#deploying-a-git-repository)。
