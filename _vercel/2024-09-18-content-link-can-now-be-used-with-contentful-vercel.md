借助 [Content Link](https://vercel.com/docs/workflow-collaboration/edit-mode#content-link)（此前称为“可视化编辑”），您可直接点击 Vercel 网站上的内容进行编辑，并一键跳转至该内容在 CMS 中的确切位置。

此功能现已面向使用 Contentful 作为其 CMS 并已订阅 Premium 套餐的 Pro 和 Enterprise 客户开放。如需查看所有受支持的 CMS 列表，请[访问我们的文档](https://vercel.com/docs/workflow-collaboration/edit-mode#content-link)。

启用该功能后，Contentful 的 API 将为某些支持可视化的字段返回 [源映射（source maps）](https://www.contentful.com/developers/docs/tools/vercel/content-source-maps-with-vercel/)，这些字段包含指向 Contentful 中对应字段的链接，例如富文本、描述字段及列表字段。目前暂不支持 Markdown。
