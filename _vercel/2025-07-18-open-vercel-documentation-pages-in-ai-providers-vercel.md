Jul 18, 2025

2025年7月18日

![](images/open-vercel-documentation-pages-in-ai-providers-vercel/img_001.jpg)![](images/open-vercel-documentation-pages-in-ai-providers-vercel/img_002.jpg)

You can now copy [Vercel documentation](https://vercel.com/docs) pages as markdown, or open them directly in [v0](https://v0.dev/), [Claude](https://claude.ai/) or [ChatGPT](https://chatgpt.com/).

现在，您可以将 [Vercel 文档](https://vercel.com/docs) 页面以 Markdown 格式复制，或直接在 [v0](https://v0.dev/)、[Claude](https://claude.ai/) 或 [ChatGPT](https://chatgpt.com/) 中打开。

This allows you to use documentation content as context when working with AI tools. Visit any documentation page and use the dropdown in the top right of the page.

这使您能在使用 AI 工具时，将文档内容作为上下文参考。访问任意文档页面，并点击页面右上角的下拉菜单。

### **Using the copy page dropdown**

### **使用“复制页面”下拉菜单**

- Navigate to any documentation page

- 访问任意文档页面

- Click the copy page dropdown in the top right corner

- 点击页面右上角的“复制页面”下拉菜单

- Select your provider or copy as markdown

- 选择目标 AI 提供商，或以 Markdown 格式复制

The page content will be formatted and loaded into the selected AI provider.

页面内容将被格式化，并自动加载至所选的 AI 提供商中。
