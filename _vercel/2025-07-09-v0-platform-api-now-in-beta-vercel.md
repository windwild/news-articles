Jul 9, 2025

2025年7月9日

![](images/v0-platform-api-now-in-beta-vercel/img_001.jpg)![](images/v0-platform-api-now-in-beta-vercel/img_002.jpg)

The v0 Platform API is now available in public beta.  

v0 平台 API 现已开放公开测试（Beta）。

The v0 Platform API is a text-to-app API — it provides programmatic access to v0’s app generation pipeline:  

v0 平台 API 是一款“文本生成应用”（text-to-app）API，它为开发者提供对 v0 应用生成流水线的程序化访问能力，包括：

- Generating code for web apps from prompts  
- 基于提示词（prompts）自动生成网页应用代码  

- Structured parsing of generated code  
- 对生成代码进行结构化解析  

- Automatic error fixing  
- 自动修复错误  

- Link with a rendered preview  
- 关联渲染后的预览界面  


This API also supports programmatic control of v0.dev, including creating and managing both chats and projects. We'll be bringing more of v0.dev's functionality into the Platform API soon.  

该 API 还支持对 v0.dev 进行程序化控制，包括创建和管理聊天会话（chats）及项目（projects）。我们将在近期把更多 v0.dev 的功能整合进平台 API 中。

The v0 Platform API is designed for integration into development workflows, automation scripts, and third-party tools.  

v0 平台 API 专为集成至开发工作流、自动化脚本及第三方工具而设计。

Check out our [TypeScript SDK](https://npmjs.com/v0-sdk) and [documentation](https://v0.dev/docs/v0-platform-api) to get started.  

欢迎查阅我们的 [TypeScript SDK](https://npmjs.com/v0-sdk) 和 [文档](https://v0.dev/docs/v0-platform-api) 开始使用。
