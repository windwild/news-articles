2023年6月15日

![](images/vercel-kv-is-now-generally-available-for-hobby-and-pro-customers-vercel/img_001.jpg)![](images/vercel-kv-is-now-generally-available-for-hobby-and-pro-customers-vercel/img_002.jpg)

The Vercel KV product has been sunset. You can now deploy alternative KV stores and other storage solutions through the [Vercel Marketplace Storage](https://vercel.com/marketplace/category/storage?category=storage&search=redis), with automatic account provisioning and unified billing. [Learn more](https://vercel.com/blog/introducing-the-vercel-marketplace).

Vercel KV 产品已正式停用。您现在可通过 [Vercel Marketplace 存储分类](https://vercel.com/marketplace/category/storage?category=storage&search=redis) 部署替代的键值（KV）存储及其他存储解决方案，享受自动账户开通与统一账单服务。[了解更多](https://vercel.com/blog/introducing-the-vercel-marketplace)。

[Vercel KV](https://vercel.com/docs/storage/vercel-kv), our durable Redis database that enables you to store and retrieve JSON data, is now generally available.

[Vercel KV](https://vercel.com/docs/storage/vercel-kv) 是我们提供的高持久性 Redis 数据库，支持您存储和检索 JSON 数据，现已全面开放使用。

This feature is available for Hobby and Pro users with Hobby users getting 1 Database, 30,000 requests per month, 256 MB total storage, and 256 MB data transfer per month. Pro users will get 1 Database, 150,000 requests per month, 512 MB total storage, 512 MB data transfer per month. **Billing will begin on June 20th.**

该功能面向 Hobby 和 Pro 用户开放：Hobby 用户可获得 1 个数据库、每月 30,000 次请求配额、总计 256 MB 存储空间及每月 256 MB 数据传输额度；Pro 用户则可获得 1 个数据库、每月 150,000 次请求配额、总计 512 MB 存储空间及每月 512 MB 数据传输额度。**计费将于 6 月 20 日起正式开始。**

On-demand pricing for Pro users has also been lowered, with total storage **reduced 17% from $0.30/GB to $0.25/GB** and data transfer **reduced 50% from $0.20/GB to $0.10/GB.**

Pro 用户的按需计价方案也已下调：存储费用 **从 $0.30/GB 降低 17%，至 $0.25/GB**；数据传输费用 **从 $0.20/GB 降低 50%，至 $0.10/GB**。

To see your usage, visit the the [Usage page](https://vercel.com/dashboard/usage) in the Dashboard. If you want to stop leveraging Vercel KV or see how to optimize your usage, you can [stop querying the database and delete it](https://vercel.com/docs/storage/vercel-kv/usage-and-pricing#managing-charges).

如需查看用量详情，请访问控制台中的 [用量页面](https://vercel.com/dashboard/usage)。若您希望停止使用 Vercel KV，或了解如何优化用量，可参考指南执行 [停止查询并删除数据库](https://vercel.com/docs/storage/vercel-kv/usage-and-pricing#managing-charges)。

If you're an Enterprise company interested in using Vercel KV, you can [contact us](https://vercel.com/contact/sales) to get started.

若您是希望采用 Vercel KV 的企业客户，欢迎 [联系我们](https://vercel.com/contact/sales) 开始合作。

[Check out the documentation](https://vercel.com/docs/storage/vercel-kv) to learn more about Vercel KV.

[查阅文档](https://vercel.com/docs/storage/vercel-kv)，深入了解 Vercel KV。
