Enterprise customers can now protect against the top OWASP risks.

企业客户现在可针对 OWASP 最高风险项提供防护。

The Vercel Firewall protects against the OWASP Core Ruleset for Enterprise, which enables Vercel to log, block, or challenge traffic against these rules.

Vercel 防火墙支持针对企业版 OWASP 核心规则集（OWASP Core Ruleset for Enterprise）进行防护，使 Vercel 能够对匹配这些规则的流量执行记录、拦截或挑战（如 CAPTCHA）操作。

In addition to [new custom rules](https://vercel.com/docs/security/vercel-waf/custom-rules), customers can also ensure they remain protected against the biggest risks for web applications with new [OWASP](https://owasp.org/) Top 10 protection. For example, this ruleset includes automatic protection against SQL injection attacks.

除新增的[自定义规则](https://vercel.com/docs/security/vercel-waf/custom-rules)外，客户还可借助全新的 [OWASP](https://owasp.org/) Top 10 防护能力，持续抵御 Web 应用面临的最主要安全风险。例如，该规则集已内置对 SQL 注入攻击的自动防护功能。
