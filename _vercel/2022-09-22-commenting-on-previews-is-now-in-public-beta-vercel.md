With comments, teams can give collaborative feedback directly on copy, components, interactions, and more right in your [Preview Deployments](https://vercel.com/features/previews).

借助评论功能，团队成员可直接在您的 [预览部署（Preview Deployments）](https://vercel.com/features/previews) 中，针对文案、组件、交互效果等开展协作式反馈。

PR owners, comment creators, and participants in comment threads can review and collaborate on real UI with comments, screenshots, notifications, all synchronized with [Slack](https://vercel.com/integrations/slack).

拉取请求（PR）所有者、评论创建者以及评论线程的参与者，均可结合评论、截图和通知，在真实 UI 环境中进行评审与协作，全部内容均与 [Slack](https://vercel.com/integrations/slack) 同步。

[Check out the documentation](https://vercel.com/docs/concepts/deployments/comments) to learn more or [opt-in](https://vercel.com/enable-comments) to start using comments now.

[查阅文档](https://vercel.com/docs/concepts/deployments/comments) 了解详情，或立即 [开启评论功能](https://vercel.com/enable-comments) 开始使用。
