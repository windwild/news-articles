![](images/search-any-public-github-repo-with-grep-vercel/img_001.jpg)![](images/search-any-public-github-repo-with-grep-vercel/img_002.jpg)

You can now use [Grep](https://grep.app/) to search _any_ public repository on GitHub, no longer limited to the 1M+ pre-indexed repos.

现在，您可使用 [Grep](https://grep.app/) 搜索 GitHub 上的_任意_公开仓库，不再局限于已预索引的 100 多万个仓库。

To search a specific repo, use `grep.app/[owner]/[repo]`.

如需搜索特定仓库，请访问 `grep.app/[所有者]/[仓库名]`。

For example: visit [grep.app/vercel/ai](https://grep.app/vercel/ai) and start typing a search query (try `streamText`).

例如：访问 [grep.app/vercel/ai](https://grep.app/vercel/ai)，然后输入搜索关键词（可尝试 `streamText`）。

Get quick, full-text and regular expression search across the repo without any setup.

无需任何配置，即可在该仓库中快速执行全文搜索和正则表达式搜索。
