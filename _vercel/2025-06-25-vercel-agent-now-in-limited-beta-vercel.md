Vercel Agent is now available in [Limited Beta](https://vercel.com/docs/release-phases#limited-beta). Agent is an AI assistant built into the Vercel dashboard that analyzes your app performance and security data.

Vercel Agent 现已开放[有限公测（Limited Beta）](https://vercel.com/docs/release-phases#limited-beta)。Agent 是内置于 Vercel 控制台的 AI 助手，可分析您应用的性能与安全数据。

Agent focuses on Observability, summarizing anomalies, identifying likely causes, and recommending specific actions. These actions can span across the platform, including managing firewall rules in response to traffic spikes or geographic anomalies, and identifying optimization opportunities within your application.

Agent 聚焦于可观测性（Observability），自动汇总异常现象、识别潜在成因，并推荐具体操作建议。这些建议覆盖平台多个层面，例如：针对流量激增或地域性异常动态调整防火墙规则，或在您的应用中识别性能优化机会。

Insights appear contextually as detailed notebooks with no configuration required.

洞察以详尽的笔记本形式上下文化呈现，无需任何配置。
