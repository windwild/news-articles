Dec 3, 2025

2025 年 12 月 3 日

Vercel Agent 现在可自动更新您的代码库，并提交一个 Pull Request（PR），为您的项目添加 [Web Analytics](https://vercel.com/docs/analytics)（网页分析）和 [Speed Insights](https://vercel.com/docs/speed-insights)（速度洞察）功能。

Vercel Agent 会分析您的项目配置及已连接的 GitHub 仓库，安装相关依赖包、插入对应的代码片段，并创建一个包含建议变更的 Pull Request。

如需使用 Vercel Agent 为您的项目安装 Web Analytics 或 Speed Insights，请按以下步骤操作：

1. 进入仪表板中的 **Analytics**（分析）或 **Speed Insights**（速度洞察）页面。

2. **启用**该功能。

3. 点击 **Implement**（实施）以启动 Vercel Agent。

4. 审阅生成的 Pull Request，并在确认无误后合并。

一旦 Pull Request 被合并并完成部署，数据追踪将自动开始。

目前，Vercel Agent 的安装功能已面向所有团队开放 **公开测试版（Public Beta）**。欢迎立即体验：[Web Analytics](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fanalytics&title=Vercel+Web+Analytics) 或 [Speed Insights](https://vercel.com/d?to=%2F%5Bteam%5D%2F%5Bproject%5D%2Fspeed-insights&title=Vercel+Speed+Insights)。
