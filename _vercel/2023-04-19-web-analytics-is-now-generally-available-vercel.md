Apr 19, 2023

2023年4月19日

![](images/web-analytics-is-now-generally-available-vercel/img_001.jpg)![](images/web-analytics-is-now-generally-available-vercel/img_002.jpg)

Vercel Web Analytics（此前名为 Audiences）现已向所有用户全面开放，可为您提供网站热门页面、主要流量来源以及用户人口统计信息（如国家/地区、操作系统和浏览器类型）等关键洞察。

Web Analytics 现在还新增了[自定义事件](https://vercel.com/docs/concepts/analytics/custom-events)追踪功能，助您更深入地分析用户交互行为，例如按钮点击、表单提交及转化事件等。

[Speed Insights](https://vercel.com/docs/concepts/speed-insights) 仍为独立产品，可通过“真实体验评分”（Real Experience Score）帮助您持续监测应用性能。

Web Analytics 在所有订阅方案中均可用：Hobby 方案每月提供 2,500 次事件配额；Pro 方案每月提供 25,000 次事件配额；Enterprise 方案则支持按需定制配额。此外，Pro 用户还可额外加购 [Web Analytics Plus](https://vercel.com/docs/concepts/analytics/limits-and-pricing)，每月仅需 50 美元，即可获得 50 万次事件配额。

[查看文档](https://vercel.com/docs/concepts/analytics) 以了解更多信息。
