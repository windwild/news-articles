![](images/you-can-now-configure-advanced-sampling-rules-for-vercel-drains-vercel/img_001.jpg)![](images/you-can-now-configure-advanced-sampling-rules-for-vercel-drains-vercel/img_002.jpg)

You can now configure advanced sampling rules when exporting data to a third-party observability tool when using Vercel Drains.

使用 Vercel Drains 将数据导出至第三方可观测性工具时，您现在可以配置高级采样规则。

Advanced sampling rules allow you to configure sampling rates for specific environments and path prefixes, providing more granular control over cost management.

高级采样规则支持您为特定环境和路径前缀分别配置采样率，从而实现更精细的成本管理控制。

Vercel Drains is available to Pro and Enterprise teams. Advanced sampling rules can be configured on drains exporting logs or traces.

Vercel Drains 面向 Pro 和 Enterprise 团队开放。高级采样规则可应用于导出日志或追踪（traces）的 Drains。

[Try it out](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fdrains) or learn more about [Vercel Drains](https://www.vercel.com/docs/drains).

[立即试用](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fdrains)，或了解有关 [Vercel Drains](https://www.vercel.com/docs/drains) 的更多信息。
