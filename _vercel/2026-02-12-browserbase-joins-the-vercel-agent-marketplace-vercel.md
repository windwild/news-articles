Feb 12, 2026

2026 年 2 月 12 日

[Browserbase](https://vercel.com/marketplace/browserbase) 现已上线 Vercel Marketplace，使团队无需自行管理基础设施，即可为 AI 智能体运行浏览器自动化任务。

该集成通过 Chrome DevTools Protocol（CDP）将智能体连接至远程浏览器，从而支持需与真实网站交互的工作流，例如登录仪表板、填写表单或浏览动态页面。

借助这一“一键式”集成，团队可享受统一计费以及专为长期、有状态会话设计的基础设施。核心能力包括：

- 仅需一个 API 密钥即可完成安装与连接  
- 通过 CDP 将智能体连接至远程浏览器  
- 降低基于浏览器的智能体工作流的运维复杂度  
- 兼容 [Vercel Sandbox](https://vercel.com/docs/vercel-sandbox) 和 [AI Gateway](https://vercel.com/ai-gateway)

同日发布的还有 [Browserbase 对 Web Bot Auth 的支持](https://vercel.com/changelog/web-bot-auth-browserbase-agents)，使智能体能够稳定访问托管于 Vercel 的部署应用，不受各类安全防护层的干扰。

立即前往 Vercel Marketplace 开始使用 [Browserbase](https://vercel.com/marketplace/browserbase)，或尝试此 [示例](https://vercel.com/templates/template/ai-research-agent)，亲身体验其实际效果。
