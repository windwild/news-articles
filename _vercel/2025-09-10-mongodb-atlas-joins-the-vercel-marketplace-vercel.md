Sep 10, 2025

2025年9月10日

![](images/mongodb-atlas-joins-the-vercel-marketplace-vercel/img_001.jpg)![](images/mongodb-atlas-joins-the-vercel-marketplace-vercel/img_002.jpg)

You can now provision [MongoDB Atlas](https://vercel.com/marketplace/mongodbatlas) directly from the Vercel Marketplace.

您现在可直接通过 Vercel Marketplace 部署 [MongoDB Atlas](https://vercel.com/marketplace/mongodbatlas)。

Spin up a fully managed MongoDB Atlas database, connect it to you Vercel project, and start building without leaving the Vercel dashboard.

快速启动一个完全托管的 MongoDB Atlas 数据库，将其连接至您的 Vercel 项目，并在不离开 Vercel 控制台的前提下立即开始开发。

This MongoDB Atlas native integration provides:

该 MongoDB Atlas 原生集成提供以下能力：

- A flexible document model for structured and unstructured data  
- 灵活的文档模型，支持结构化与非结构化数据  

- Built-in search, including vector and semantic search  
- 内置搜索功能，包括向量搜索与语义搜索  

- Horizontal scaling with replica sets and sharding  
- 借助副本集（replica sets）与分片（sharding）实现水平扩展  

- Free, pre-provisioned, or serverless deployment options  
- 免费、预配置或无服务器（serverless）等多种部署选项  

This integration removes the friction of switching between dashboards or managing complex setup, giving developers a fast, modern data layer to power web and AI applications on Vercel.

该集成消除了在多个控制台间切换或管理复杂配置所带来的障碍，为开发者提供了一套快速、现代化的数据层，助力其在 Vercel 上构建 Web 应用与 AI 应用。

在 Vercel Marketplace 上开始使用 [MongoDB Atlas](https://vercel.com/new/clone?demo-description=Test+template+for+native+Mongo+integration.&demo-image=%2F%2Fimages.ctfassets.net%2Fe5382hct74si%2F1C5qjoXApiKj8hFdKy55bp%2Fbcf64203cf35dccf71e0a59f7535876d%2Fhome.png&demo-title=Vercel+with+Mongo&demo-url=https%3A%2F%2Fvercel-marketplace-mongo.vercel.app%2F&from=templates&products=%255B%257B%2522integrationSlug%2522%253A%2522mongodbatlas%2522%252C%2522productSlug%2522%253A%2522atlas%2522%252C%2522protocol%2522%253A%2522storage%2522%252C%2522type%2522%253A%2522integration%2522%257D%255D&project-name=Vercel+with+Mongo&repository-name=nextjs-template-mongodb&repository-url=https%3A%2F%2Fgithub.com%2Fmongodb-developer%2Fnextjs-template-mongodb&skippable-integrations=1)，所有套餐的客户均可使用。
