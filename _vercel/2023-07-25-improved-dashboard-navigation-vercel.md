2023年7月25日

![](images/improved-dashboard-navigation-vercel/img_001.jpg)![](images/improved-dashboard-navigation-vercel/img_002.jpg)

The dashboard navigation has received a visual update. You can now see the project icon in the navigation and the mobile version only shows the name of the scope you are currently on to save space.

仪表板导航已进行视觉更新。现在，您可以在导航栏中看到项目图标；而在移动设备版本中，为节省空间，仅显示您当前所在作用域的名称。

Check out the [documentation](https://vercel.com/docs/concepts/dashboard-features) to learn more.

请查阅[文档](https://vercel.com/docs/concepts/dashboard-features)了解更多信息。
