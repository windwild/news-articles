Nov 21, 2025

2025 年 11 月 21 日

[Vercel Agent investigations](https://vercel.com/docs/agent/investigation) are now included in Observability Plus, adding 10 investigations to every billing cycle at no extra cost to your subscription.

[Vercel Agent 调查功能](https://vercel.com/docs/agent/investigation) 现已包含在 Observability Plus（可观测性增强版）中，每个计费周期免费提供 10 次调查，无需额外付费。

Investigations help teams diagnose and resolve incidents faster, and run automatically on error alerts. When an alert flags suspicious activity, such as unexpected spikes in usage or errors, Vercel Agent investigates the issue, identifies the likely root cause, analyzes the impact, and suggests next steps for remediation.

调查功能可帮助团队更快地诊断和解决事件，并在触发错误告警时自动运行。当告警标记出可疑活动（例如用量或错误率出现意外激增）时，Vercel Agent 将自动开展调查，识别最可能的根本原因，分析影响范围，并提出后续修复建议。

Teams can purchase Vercel Agent [credits](https://vercel.com/docs/agent/pricing) to run additional investigations. Investigations are public beta for Pro and Enterprise teams with [Observability Plus](https://vercel.com/docs/observability/observability-plus).

团队可通过购买 Vercel Agent [积分](https://vercel.com/docs/agent/pricing) 来运行更多次调查。该调查功能目前面向已订阅 [Observability Plus](https://vercel.com/docs/observability/observability-plus) 的 Pro 和 Enterprise 团队开放公开测试（Public Beta）。

[Try it out](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fvercel-agent) or learn more about [Vercel Agent investigations](https://vercel.com/docs/agent/investigation).

[立即试用](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fvercel-agent)，或深入了解 [Vercel Agent 调查功能](https://vercel.com/docs/agent/investigation)。
