2025 年 2 月 27 日

![](images/ip-address-details-added-in-the-vercel-firewall-dashboard-vercel/img_001.jpg)![](images/ip-address-details-added-in-the-vercel-firewall-dashboard-vercel/img_002.jpg)

The Vercel Firewall dashboard now displays enriched IP address data, including the [autonomous system](https://en.wikipedia.org/wiki/Autonomous_system_(Internet)) (AS) name, AS number (ASN), and geolocation on hover.

Vercel 防火墙（Firewall）控制台现已支持在悬停时显示更丰富的 IP 地址信息，包括[自治系统](https://zh.wikipedia.org/wiki/%E8%87%AA%E6%B2%BB%E7%B3%BB%E7%BB%9F_(%E4%BA%92%E8%81%94%E7%BD%91))（Autonomous System，AS）名称、AS 编号（ASN）以及地理位置信息。

This information helps identify the origin of an attack, determine the owner of an IP address, and create [targeted custom rules](https://vercel.com/docs/security/vercel-waf/custom-rules) to block malicious traffic.

这些信息有助于识别攻击来源、确认 IP 地址所属方，并创建[精准的自定义规则](https://vercel.com/docs/security/vercel-waf/custom-rules)，以拦截恶意流量。

Learn more about the [Vercel Firewall](https://vercel.com/docs/security/vercel-firewall).

了解更多关于 [Vercel 防火墙](https://vercel.com/docs/security/vercel-firewall) 的信息。
