Aug 26, 2021

2021年8月26日

![](images/request-access-to-teams-right-from-the-dashboard-vercel/img_001.jpg)

要加入 Vercel 上的一个 Team，该 Team 的任意一位所有者均可通过仪表板中的 Team 设置邀请您加入（或者，在启用 [SAML](https://vercel.com/changelog/saml-single-sign-on-and-directory-sync-now-fully-available) 的情况下，将其添加至相应的目录同步（Directory Sync）服务提供商中）。

此外，**平台现在还会为您推荐可申请加入的 Teams**，推荐依据包括：

- 您个人账户所关联的任意 Git 登录连接（Git Login Connections）有权访问的 Git 命名空间（例如 GitHub 组织）。

- 与您的个人账户相关联的电子邮件域名。

推荐的 Teams 将显示在您个人账户设置中的一个**全新 Teams 页面**上（该页面同时也汇总展示了您当前已加入的所有 Teams），以及仪表板左上方的作用域选择器（scope selector）中。

也可参阅[相关文档](https://vercel.com/docs/platform/users-and-teams#suggested-teams)。
