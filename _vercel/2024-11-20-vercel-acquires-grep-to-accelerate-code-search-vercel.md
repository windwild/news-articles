Nov 20, 2024

2024 年 11 月 20 日

宣布收购 Grep，以进一步推进我们“助力开发者更高效地开发与交付”的使命。

[Grep](https://grep.app/) 使开发者能够快速在超过 50 万个公开 Git 仓库中搜索代码。此次收购后，Grep 创始人 Dan Fox 将加入 Vercel 的 AI 团队，继续推进 Grep 的开发，以增强面向开发者的代码搜索能力。

![](images/vercel-acquires-grep-to-accelerate-code-search-vercel/img_001.jpg)

## 快速定位所需内容

Grep 可助您：

- **更快调试**：通过搜索错误信息，精准定位其抛出位置；  
- **更快学习**：查找您正尝试使用的新函数的实际调用示例；  
- **更快上手开发**：检索与需求高度相关的代码片段。

假设您正在编写 React 代码，希望根据表单操作（form action）的结果来更新状态。您注意到 React 提供了一个名为 `useActionState` 的 Hook，看起来正好满足您的需求，但尚不确定具体用法。此时，您只需在 Grep 中输入 `useActionState`，即可迅速浏览数百个与您需求高度相似的使用示例。
