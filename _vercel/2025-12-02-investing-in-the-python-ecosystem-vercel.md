Dec 2, 2025

2025 年 12 月 2 日

The team behind [Gel Data](https://www.geldata.com/) is joining Vercel to help us invest in the Python ecosystem. Led by Python core developer [Yury Selivanov](https://x.com/1st1) and contributor [Elvis Pranskevichus](https://x.com/elprans), they will bring world-class support for Python on the AI Cloud.

[Gel Data](https://www.geldata.com/) 团队将加入 Vercel，助力我们加大对 Python 生态系统的投入。该团队由 Python 核心开发者 [Yury Selivanov](https://x.com/1st1) 和贡献者 [Elvis Pranskevichus](https://x.com/elprans) 领衔，将为 AI Cloud 提供世界级的 Python 支持。

## A commitment to the Python community

## 致力于 Python 社区

Python is one of the most widely used programming languages in the world, powering complex backends, ML pipelines, data infrastructure, and increasingly, AI. Our customers want strong Python support on Vercel, and as we continue to build the AI Cloud, we want Python to be better for everyone. So we're investing in the ecosystem itself:

Python 是全球使用最广泛的编程语言之一，驱动着复杂的后端系统、机器学习流水线、数据基础设施，并日益成为人工智能开发的核心语言。我们的客户期待 Vercel 提供强大的 Python 支持；而随着我们持续推进 AI Cloud 的建设，我们也希望 Python 能为每一位开发者带来更佳体验。因此，我们将直接投资于整个生态系统：

- [**Python Software Foundation**](https://www.python.org/psf-landing/) **Sponsorship:** We're joining as a Maintaining-level sponsor, which includes presence at [PyCon US](https://us.pycon.org/2026/) and support for the PSF's mission to advance the Python language and community.

- [**Python 软件基金会（PSF）**](https://www.python.org/psf-landing/) **赞助计划：** 我们将以“维护级（Maintaining-level）”赞助商身份加入，包括在 [PyCon US 2026](https://us.pycon.org/2026/) 大会上亮相，并支持 PSF 推动 Python 语言发展与社区建设的使命。

- **Sponsoring a Python core maintainer:** We're committing to sponsor [Serhiy Storchaka](https://x.com/SerhiyStorchaka) for one year through the Python Software Foundation. Serhiy is one of the top three Python core developers by commits over the past decade, with contributions spanning the core interpreter, standard library improvements, and performance optimizations.

- **资助一名 Python 核心维护者：** 我们将通过 Python 软件基金会，为期一年资助 [Serhiy Storchaka](https://x.com/SerhiyStorchaka)。过去十年中，Serhiy 是按代码提交量排名前三的 Python 核心开发者之一，其贡献涵盖 Python 解释器核心、标准库改进以及性能优化等多个关键领域。

- **Conference and community support:** We're committing to sponsor key Python conferences and local meetups, helping ensure the community has the resources and spaces to gather, build, and learn. We're planning our first Vercel + Python hackathon in San Francisco in February. Details are coming soon, so keep an eye out on [our events calendar](https://community.vercel.com/c/events/42).

- **会议与社区支持：** 我们承诺资助一系列重要 Python 技术大会及本地线下聚会，助力社区获得充足资源与交流空间，以促进协作、共建与学习。我们计划于二月在旧金山举办首届 Vercel × Python 黑客松（Hackathon）。更多详情即将公布，请持续关注我们的 [活动日历](https://community.vercel.com/c/events/42)。

## The team behind uvloop and asyncpg

## uvloop 与 asyncpg 背后的团队

Yury is the creator of [uvloop](https://github.com/MagicStack/uvloop), the high-performance event loop for Python that's downloaded 3 million times per day. It's the foundation beneath [FastAPI](https://fastapi.tiangolo.com/) and many other Python applications that need speed and reliability. He also contributed async/await syntax to Python along with seven other Python enhancement proposals (PEPs) and influenced the design of asyncio since before it was part of the Python standard library.

Yury 是 [uvloop](https://github.com/MagicStack/uvloop) 的作者——这是一个高性能 Python 异步事件循环，日均下载量达 300 万次。它已成为 [FastAPI](https://fastapi.tiangolo.com/) 及众多对性能与可靠性有严苛要求的 Python 应用程序的底层基石。此外，他还参与设计了 Python 的 `async`/`await` 语法，并主导或联合提出了七项其他 Python 增强提案（PEP），早在 `asyncio` 被纳入 Python 标准库之前，便已深度影响其整体架构设计。

Elvis and Yury also created [asyncpg](https://github.com/MagicStack/asyncpg), the PostgreSQL library that's deeply embedded in the Python ecosystem.

Elvis 和 Yury 还共同创建了 [asyncpg](https://github.com/MagicStack/asyncpg)，这是一款深度融入 Python 生态系统的 PostgreSQL 库。

Together, they've spent years pushing the ecosystem forward for developers worldwide. They'll continue contributing to open source Python projects while helping us build strong support for Python developers on Vercel.

多年来，他们一直致力于推动全球开发者所依赖的 Python 生态系统向前发展。未来，他们将继续为开源 Python 项目贡献力量，同时协助我们为 Vercel 上的 Python 开发者构建坚实的支持体系。

## What this means for Vercel

## 这对 Vercel 意味着什么

As we continue building infrastructure for the next generation of web applications and AI agents, we're expanding beyond our JavaScript and TypeScript roots.

在持续为下一代 Web 应用与 AI 智能体构建基础设施的过程中，我们正逐步突破以 JavaScript 和 TypeScript 为根基的传统边界。

Over the coming months, we'll be investing in Python support across Vercel. This includes:

在未来数月内，我们将在 Vercel 全面加大 Python 支持投入。具体包括：

- Improving Python framework support  
- 提升对 Python 框架的支持能力  

- Making Python deployment as simple and fast as it is for JavaScript frameworks  
- 让 Python 部署体验与 JavaScript 框架一样简单、快速  

- Contributing back to the tools and libraries the community relies on  
- 向社区广泛依赖的工具和库回馈贡献  

We're taking a "building in public" approach. As we ship improvements to Python support on Vercel, we'll share what we're learning and how we're approaching these problems. We also want to hear your feedback as we go. This is just the beginning, and we can’t wait to see what Yury, Elvis, and the team will build.

我们采用“公开共建”（building in public）的方式：每当我们在 Vercel 上推出 Python 支持的新改进，都将同步分享我们的实践心得与问题解决思路；我们也热切期待您在过程中提出的反馈意见。这一切才刚刚开始，我们无比期待 Yury、Elvis 及整个团队将创造出怎样的成果。

Gel Data, the Postgres-based data platform, will wind down on January 31, 2026. We're not commercializing the platform, but the open source components will remain available. This acquisition is focused on Python expertise and community investment, not entering the database market. We partner with excellent database providers via the Vercel Marketplace, and we have no plans for that to change.

基于 PostgreSQL 的数据平台 Gel Data 将于 2026 年 1 月 31 日正式停止运营。我们不会对该平台进行商业化运作，但其所有开源组件将持续对外可用。本次收购的核心目标是获取 Python 领域的专业能力并加大对 Python 社区的投入，而非进军数据库市场。我们已通过 Vercel Marketplace 与多家优秀的数据库服务商建立合作关系，这一策略亦不会发生改变。

We aimed to challenge the state of the art when we founded MagicStack and then Gel, and we will continue to push the envelope with this new support. ## Building for the long term

## 着眼长远，持续建设

This is the first step in our commitment to Python. We're here to contribute to the ecosystem, support the people who make it great, and help make Python more powerful.

这是我们在 Python 领域长期承诺迈出的第一步。我们致力于为整个生态贡献力量，支持那些让 Python 如此卓越的开发者，并助力 Python 变得更加强大。

We believe the best infrastructure is built on [open foundations](https://vercel.com/blog/open-sdk-strategy). Whether it's JavaScript frameworks like Next.js, Nuxt, and SvelteKit, or now Python tools and libraries, our role is to support what developers already love and make it better.

我们坚信，最优秀的基础设施应建立在[开放的基础之上](https://vercel.com/blog/open-sdk-strategy)。无论是 Next.js、Nuxt 和 SvelteKit 等 JavaScript 框架，还是如今的 Python 工具与库，我们的使命始终是：支持开发者真正喜爱的技术，并将其打磨得更加出色。

We're honored to welcome Yury, Elvis, and the Gel Data team to Vercel. And we're excited to start building with the Python community. For the open web.

我们深感荣幸，欢迎尤里（Yury）、埃尔维斯（Elvis）以及 Gel Data 团队加入 Vercel。我们亦满怀期待，即将携手 Python 社区开启共建之旅——为开放的网络而构建。
