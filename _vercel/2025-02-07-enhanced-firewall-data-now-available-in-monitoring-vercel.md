Feb 7, 2025

2025 年 2 月 7 日

![](images/enhanced-firewall-data-now-available-in-monitoring-vercel/img_001.jpg)![](images/enhanced-firewall-data-now-available-in-monitoring-vercel/img_002.jpg)

Monitoring now has better firewall support, offering insights into your firewall rules:

监控功能现已增强防火墙支持，可为您提供有关防火墙规则的深入洞察：

- Filter blocked requests by actions and [custom firewall rules](https://vercel.com/docs/security/vercel-waf/custom-rules)

  - 按操作类型及[自定义防火墙规则](https://vercel.com/docs/security/vercel-waf/custom-rules)筛选被拦截的请求

- More fields are now displayed when available:

  - IP Country  
  - User Agent  
  - Route  
  - Request Path  
  - Region  

- 在数据可用时，将显示更多字段：  
  - IP 所属国家/地区  
  - 用户代理（User Agent）  
  - 路由（Route）  
  - 请求路径（Request Path）  
  - 区域（Region）

These metrics are available for all [Observability Plus](https://vercel.com/docs/observability) and [Monitoring](https://vercel.com/docs/observability/monitoring) customers.

这些指标面向所有 [Observability Plus](https://vercel.com/docs/observability) 和 [Monitoring](https://vercel.com/docs/observability/monitoring) 客户提供。

Monitoring recently became [part of Observability Plus](https://vercel.com/changelog/monitoring-pricing-reduced-up-to-87).

监控功能近期已[成为 Observability Plus 的一部分](https://vercel.com/changelog/monitoring-pricing-reduced-up-to-87)。
