The Vercel Firewall now has a REST API, with the ability to:

Vercel 防火墙现已提供 REST API，支持以下功能：

- Read the current Firewall configuration  
  读取当前防火墙配置

- Create and manage custom rules  
  创建并管理自定义规则

- Enable or disable Attack Challenge Mode  
  启用或禁用攻击挑战模式（Attack Challenge Mode）

- Control and manage a list of blocked IP addresses  
  控制并管理被屏蔽的 IP 地址列表

Learn more and get started today with the [REST API documentation](https://vercel.com/docs/security/vercel-waf/examples#using-the-firewall-rest-api).

请通过 [REST API 文档](https://vercel.com/docs/security/vercel-waf/examples#using-the-firewall-rest-api) 了解更多信息，并立即开始使用。
