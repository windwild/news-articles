## **Deploy and deliver content faster at scale**

## **大规模更快地部署与交付内容**

Next.js is designed to meet you where you are and grow with you. At Next.js Conf, hear developers share lessons learned as they scale their applications and modernize their architectures.

Next.js 的设计理念是“始于当下，伴你成长”——无论您当前处于何种开发阶段，它都能无缝支持并随您的业务一同演进。在 Next.js Conf 上，您将听到众多开发者分享他们在规模化扩展应用、现代化架构升级过程中的宝贵经验与实践心得。

![](images/next_js-conf-2022-iterate-scale-and-deliver-a-great-ux-vercel/img_002.jpg)

![](images/next_js-conf-2022-iterate-scale-and-deliver-a-great-ux-vercel/img_002.jpg)

[Scaling Next.js with WordPress to a million developers](https://nextjs.org/conf/speakers/mrahmadawais)

[借助 WordPress 将 Next.js 规模化至百万开发者](https://nextjs.org/conf/speakers/mrahmadawais)

RapidAPI’s VP of DevRel Ahmad Awais chose to augment WordPress with Next.js and Incremental Static Regeneration (ISR) to get the best of both worlds—and scale to over one million developers. You’ll finish this talk understanding [how to combine WordPress with Next.js](https://vercel.com/guides/wordpress-with-vercel) for a powerful combination that enables massive scale and growth.

RapidAPI 的开发者关系副总裁 Ahmad Awais 选择将 WordPress 与 Next.js 及增量静态再生（ISR）相结合，从而兼收二者之长——最终实现面向逾百万开发者的规模化扩展。听完本场分享，您将深入理解[如何将 WordPress 与 Next.js 协同使用](https://vercel.com/guides/wordpress-with-vercel)，打造强大组合，支撑海量规模与持续增长。

[Data that scales: from modern frontends to the edge](https://nextjs.org/conf/speakers/codestackr)

[可弹性伸缩的数据：从现代前端到边缘计算](https://nextjs.org/conf/speakers/codestackr)

Tune in to learn how to deliver highly scalable, globally distributed applications with ease. This talk covers how MongoDB’s developer data platform integrates seamlessly with Next.js and Vercel by simplifying your development experience with an intuitive document model and built-in data services.

敬请收看，了解如何轻松构建高度可扩展、全球分布式部署的应用程序。本场分享将介绍 MongoDB 开发者数据平台如何与 Next.js 和 Vercel 无缝集成——通过直观的文档模型与内置数据服务，显著简化您的开发体验。

[How Next.js & Prisma make frontend developers full stack](https://nextjs.org/conf/speakers/selfteachme)

Tru Narla, Discord software engineer and popular streamer, will go over the logistics of how to set up a design system in a Next.js application, break the Figma components out into React components, handle type safety, and build accessible and easily customizable UI.

Tru Narla，Discord 软件工程师、知名直播主，将详细介绍如何在 Next.js 应用中搭建设计系统：包括将 Figma 组件转化为 React 组件、保障类型安全，以及构建具备无障碍访问能力且易于定制的用户界面。

## **Advanced experimentation with Next.js and Edge Middleware**

## **借助 Next.js 与边缘中间件实现高级实验功能**

Experimentation is crucial to developing a robust user experience, but it often comes at the expense of performance. Join developers from Tinybird and Builder.io as they explore how Next.js can deliver great user experiences, without sacrificing performance.

实验是打造稳健用户体验的关键环节，但往往以牺牲性能为代价。欢迎加入 Tinybird 与 Builder.io 的开发者团队，共同探讨 Next.js 如何在不妥协性能的前提下，交付卓越的用户体验。

![](images/next_js-conf-2022-iterate-scale-and-deliver-a-great-ux-vercel/img_007.png)

[Building an extensible web analytics SaaS in 3 weeks with Next.js](https://nextjs.org/conf/speakers/raqyuste)

[使用 Next.js 在三周内构建可扩展的 Web 分析 SaaS 平台](https://nextjs.org/conf/speakers/raqyuste)

Tinybird Frontend Engineer Raquel Yuste will share how her team built a fully-fledged and extensible analytics tracker and dashboard using [Vercel Edge Middleware](https://vercel.com/blog/vercel-edge-middleware-dynamic-at-the-speed-of-static) in just three weeks. Tune in to see how [Edge Functions](https://vercel.com/features/edge-functions) enabled the build and how developers can extend the real-time analytics open-source starterkit with Next.js, Tinybird, and Vercel—all using a privacy-first approach.

Tinybird 前端工程师 Raquel Yuste 将分享她的团队如何仅用三周时间，借助 [Vercel Edge Middleware](https://vercel.com/blog/vercel-edge-middleware-dynamic-at-the-speed-of-static)，构建出功能完备、高度可扩展的分析追踪器与数据看板。敬请关注本次分享，了解 [Edge Functions](https://vercel.com/features/edge-functions) 如何赋能该架构的快速实现，以及开发者如何基于 Next.js、Tinybird 和 Vercel 扩展这一实时分析开源 Starter Kit——全程贯彻以隐私为先的设计理念。

![](images/next_js-conf-2022-iterate-scale-and-deliver-a-great-ux-vercel/img_008.jpg)

![](images/next_js-conf-2022-iterate-scale-and-deliver-a-great-ux-vercel/img_008.jpg)

[Optimally using third-party services with Next.js](https://nextjs.org/conf/speakers/builderio)

[在 Next.js 中最优地集成第三方服务](https://nextjs.org/conf/speakers/builderio)

Builder.io Co-Founder and CEO Steve Sewell will describe how to load third-party scripts with Next.js so you can have a fast, feature-rich, and always up-to-date Next.js site. You'll learn how and when to use the Script component, how to maximize data-freshness, and more.

Builder.io 联合创始人兼首席执行官 Steve Sewell 将讲解如何在 Next.js 中高效加载第三方脚本，从而打造一个兼具高性能、丰富功能且始终同步更新的 Next.js 网站。您将学到 Script 组件的使用场景与最佳实践、如何最大限度保障数据新鲜度等关键技巧。
