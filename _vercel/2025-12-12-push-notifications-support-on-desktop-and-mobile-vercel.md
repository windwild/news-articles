Push notifications are now available on both desktop and mobile, with support for all notification types.

推送通知现已同时支持桌面端和移动端，并全面兼容所有通知类型。

To start receiving push notifications from Vercel:

如需开始接收来自 Vercel 的推送通知，请按以下步骤操作：

- Go to [Notification Settings](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fnotifications%3Futm_source%3Dchangelog%26utm_medium%3Dweb&title=Go+to+Notification+Settings) in the Vercel dashboard

- 前往 Vercel 控制台中的 [通知设置](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fnotifications%3Futm_source%3Dchangelog%26utm_medium%3Dweb&title=Go+to+Notification+Settings)

- Enable the `push` notification channel for any notification type

- 为任意通知类型启用 `push`（推送）通知渠道

To allow mobile notifications on your phone:

如需在手机上启用移动推送通知，请执行以下操作：

- Open the Vercel Dashboard in your mobile browser

- 在手机浏览器中打开 Vercel 控制台

- Opt in to push notifications when prompted

- 当系统提示时，允许接收推送通知

[Try it out](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fnotifications%3Futm_source%3Dchangelog%26utm_medium%3Dweb&title=Go+to+Notification+Settings) or learn more about [notifications](https://vercel.com/docs/notifications).

[立即尝试](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fsettings%2Fnotifications%3Futm_source%3Dchangelog%26utm_medium%3Dweb&title=Go+to+Notification+Settings)，或深入了解 [通知功能](https://vercel.com/docs/notifications)。
