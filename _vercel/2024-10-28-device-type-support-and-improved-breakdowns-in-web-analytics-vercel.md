2024年10月28日

![](images/device-type-support-and-improved-breakdowns-in-web-analytics-vercel/img_001.jpg)![](images/device-type-support-and-improved-breakdowns-in-web-analytics-vercel/img_002.jpg)

You can now inspect and filter device types in [Vercel Web Analytics](https://vercel.com/docs/analytics), and apply filters to view page views and custom events for each device.

现在，您可以在 [Vercel Web Analytics](https://vercel.com/docs/analytics) 中查看并筛选设备类型，并应用筛选器来分别查看各设备类型的页面浏览量和自定义事件。

Additionally, we updated the overview by showing percentages instead of absolute numbers to see the overall distribution. You can still explore the total numbers when expanding a certain panel by clicking the "View all" button.

此外，我们更新了概览视图，以百分比（而非绝对数值）展示数据，以便更清晰地了解整体分布情况。您仍可通过点击“查看全部”（View all）按钮展开特定面板，查看具体的总数。

These features are available to Web Analytics users on all plans.

上述功能面向所有订阅计划的 Web Analytics 用户开放。

Check out [our documentation](https://vercel.com/docs/analytics/filtering#using-filters) to learn more.

请查阅[我们的文档](https://vercel.com/docs/analytics/filtering#using-filters)，了解更多信息。
