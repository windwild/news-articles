Dec 5, 2023

2023 年 12 月 5 日

![](images/conformance-and-code-owners-are-now-generally-available-for-enterprise-teams-ver/img_001.jpg)![](images/conformance-and-code-owners-are-now-generally-available-for-enterprise-teams-ver/img_002.jpg)

Today, we're releasing new features to Vercel's [Developer Experience Platform](https://vercel.com/products/dx-platform) to help [Enterprise teams](https://vercel.com/enterprise) ship higher quality code, faster—even as teams and codebases scale:

今天，我们面向 Vercel 的 [开发者体验平台（Developer Experience Platform）](https://vercel.com/products/dx-platform) 推出多项新功能，助力 [企业团队（Enterprise teams）](https://vercel.com/enterprise) 更快速地交付更高质量的代码——即使团队规模与代码库持续扩大。

[**Conformance**](https://vercel.com/docs/workflow-collaboration/conformance) **:** Maintain high-quality code standards across projects in your codebase.

[**合规性（Conformance）**](https://vercel.com/docs/workflow-collaboration/conformance)：在您的整个代码库中跨项目统一维护高标准的代码质量。

- **Conformance CLI:** Run Conformance in your CI/CD systems to block the merge of new code, or run it locally to catch issues before even committing them.

- **Conformance CLI**：在 CI/CD 系统中运行 Conformance，以阻止不符合规范的新代码合入；或在本地运行，在提交前即发现潜在问题。

- **Custom Rules:** Add organization-specific rules to ensure codebase consistency.

- **自定义规则（Custom Rules）**：添加组织专属的检查规则，确保代码库风格与实践的一致性。

[**Code Owners**](https://vercel.com/docs/workflow-collaboration/code-owners) **:** Integrate with your Git client for streamlined code reviews and smart reviewer assignments.

[**代码负责人（Code Owners）**](https://vercel.com/docs/workflow-collaboration/code-owners)：与您的 Git 客户端深度集成，简化代码评审流程，并实现智能评审人自动分配。

- **Reviewer Assignments**: Intelligent code review assignments based on your organization's structure.

- **评审人自动分配（Reviewer Assignments）**：基于组织架构，智能分配代码评审任务。

- **Modifiers**: Customize your review process to fit your team's needs. Assign reviews in a round-robin style, based on who's on call, or to the whole team.

- **修饰符（Modifiers）**：灵活定制评审流程，满足团队实际需求。例如，可按轮值（round-robin）方式、根据当前值班人员（on-call），或面向整个团队分配评审任务。

[**A reimagined dashboard experience**](https://vercel.com/docs/workflow-collaboration/conformance/dashboard-overview) **:** When you start using Conformance, you’ll see a reengineered workspace to surface code health insights, aid cross-team collaboration, and ensure a better onboarding experience for new team members.

[**焕然一新的仪表板体验**](https://vercel.com/docs/workflow-collaboration/conformance/dashboard-overview)：启用 Conformance 后，您将看到一个全面重构的工作区界面，用以直观呈现代码健康度洞察、促进跨团队协作，并为新成员提供更顺畅的上手体验。

[Check out the documentation](https://vercel.com/docs/workflow-collaboration/conformance) to learn more or [contact us](https://vercel.com/contact/sales) to get started.

请查阅[文档](https://vercel.com/docs/workflow-collaboration/conformance)了解更多信息，或[联系我们](https://vercel.com/contact/sales)立即开始。
