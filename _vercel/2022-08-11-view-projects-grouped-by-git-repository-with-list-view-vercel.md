2022年8月11日

![](images/view-projects-grouped-by-git-repository-with-list-view-vercel/img_001.jpg)![](images/view-projects-grouped-by-git-repository-with-list-view-vercel/img_002.jpg)

You can now view projects on the dashboard grouped by their repository with list view.

您现在可以在仪表板上以列表视图查看项目，并按其 Git 仓库进行分组。

List view improves the experience for teams using monorepos or a large number of projects. Projects are sorted by date and displayed as a list. You can use the toggle to switch between the card or list view for displaying projects, with your preference saved across devices.

列表视图显著提升了使用单体仓库（monorepo）或管理大量项目的团队的使用体验。项目将按创建或更新日期排序，并以列表形式展示。您可通过切换按钮在卡片视图与列表视图之间自由切换，您的偏好设置将在所有设备间同步保存。

Check out the [documentation](https://vercel.com/docs/dashboard-features/overview) to learn more.

请查阅[文档](https://vercel.com/docs/dashboard-features/overview)，了解更多信息。
