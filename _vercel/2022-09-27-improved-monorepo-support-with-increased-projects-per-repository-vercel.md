![](images/improved-monorepo-support-with-increased-projects-per-repository-vercel/img_001.jpg)![](images/improved-monorepo-support-with-increased-projects-per-repository-vercel/img_002.jpg)

To help your monorepo grow, we have updated the number of projects that you are able to add from a single git repository for both Pro and Enterprise plans.

为助力您的单体仓库（monorepo）持续扩展，我们已更新 Pro 和 Enterprise 计划下，从单个 Git 仓库可关联的项目数量上限。

Pro users can attach up to `60` (increased from `10`) projects per single git repository, and the Enterprise limit has more than doubled.

Pro 用户现可从单个 Git 仓库关联最多 `60` 个项目（此前上限为 `10`）；Enterprise 用户的限额则提升至原先的两倍以上。

Check out the [documentation](https://vercel.com/docs/concepts/limits/overview) to learn more.

请查阅 [文档](https://vercel.com/docs/concepts/limits/overview) 了解详情。
