The Vercel Marketplace has a new **Video** category for tools that allow developers to integrate video functionality into any project.

Vercel Marketplace 新增了一个 **视频（Video）** 类别，专门收录可帮助开发者将视频功能集成至任意项目的工具。

The first integration in the **Video** category is [Mux](https://www.mux.com/blog/mux-video-vercel-marketplace), an API-first platform for video. **视频（Video）** 类别中的首个集成是 [Mux](https://www.mux.com/blog/mux-video-vercel-marketplace)，一个以 API 为先的视频平台。借助官方推出的 [Mux 集成](https://vercel.com/new/clone?demo-description=A+template+for+building+video+uploading+%26+playback+with+Mux+Video+%26+Next.js.&demo-image=%2F%2Fimages.ctfassets.net%2Fe5382hct74si%2F4ir3RQ5tBjZB2fFyOsMJ0G%2Fc3509af8f1d02e6434e22ddadb75844d%2FCleanShot_2022-07-15_at_13.24.47.png&demo-title=Mux+Video+Starter&demo-url=https%3A%2F%2Fwith-mux-video.vercel.app%2F&from=templates&project-name=Mux+Video+Starter&repository-name=mux-video&repository-url=https%3A%2F%2Fgithub.com%2Fvercel%2Fnext.js%2Ftree%2Fcanary%2Fexamples%2Fwith-mux-video&skippable-integrations=1&products=%5B+%7B+%22type%22%3A+%22integration%22%2C+%22productSlug%22%3A+%22mux%22%2C+%22integrationSlug%22%3A+%22mux%22%2C+%22protocol%22%3A%22video%22+%7D+%5D)，Vercel 用户可以：

- Add video streaming and playback capabilities with minimal setup  
- 仅需极少配置，即可添加视频流式传输与播放功能  

- Access real-time video performance data and analytics  
- 实时访问视频性能数据与分析报告  

- Leverage integrated billing through Vercel  
- 通过 Vercel 实现账单一体化管理  

Get started with [Mux on the Vercel Marketplace](https://vercel.com/marketplace/mux), available to customers on all plans.

立即通过 [Vercel Marketplace 上的 Mux](https://vercel.com/marketplace/mux) 开始使用——所有订阅方案的客户均可使用。
