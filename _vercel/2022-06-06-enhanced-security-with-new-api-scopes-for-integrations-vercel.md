Vercel Integrations now have improved API scopes to allow or restrict access to specific features of your Vercel account.

Vercel 集成现已支持增强的 API 范围（API Scopes），可精确授予或限制对您 Vercel 账户中特定功能的访问权限。

All new integrations are required to select which API scopes they need access to and existing integrations are required to add scopes by **July 31, 2022**.

所有新创建的集成均须明确选择其所需访问的 API 范围；现有集成则须于 **2022 年 7 月 31 日前** 补充配置相应范围。

Check out [the documentation](https://vercel.com/docs/integrations/reference#using-the-vercel-api/scopes) as well.

也欢迎查阅 [相关文档](https://vercel.com/docs/integrations/reference#using-the-vercel-api/scopes)。
