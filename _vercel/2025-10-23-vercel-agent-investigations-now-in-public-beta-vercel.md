![](images/vercel-agent-investigations-now-in-public-beta-vercel/img_001.jpg)![](images/vercel-agent-investigations-now-in-public-beta-vercel/img_002.jpg)

[Vercel Agent](https://vercel.com/docs/agent/overview) 现在可针对 [异常告警](https://vercel.com/docs/alerts) 执行 AI 调查，协助团队更快地诊断与解决事故。AI 调查可简化事故响应流程、提升生产环境稳定性，并缓解告警疲劳，从而加快团队的交付速度。

当异常告警检测到可疑活动（例如用量或错误率出现意外激增）时，Vercel Agent 将自动开展调查，识别最可能的根本原因、分析影响范围，并为修复工作提供建议步骤。

如需更强的控制能力，您还可直接在异常告警详情页中手动触发 AI 调查。

Vercel Agent 的调查功能现已面向启用 [Observability Plus](https://vercel.com/docs/observability/observability-plus) 的 Pro 与 Enterprise 团队开放公开测试（Public Beta）。该功能按使用量计费，团队注册后可获赠 $100 的 Vercel Agent [积分](https://vercel.com/docs/agent/overview#pricing)，用于快速上手。

[立即试用](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fvercel-agent) 或进一步了解 [Vercel Agent 调查功能](https://vercel.com/docs/agent/investigation)。
