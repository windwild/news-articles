2023年8月23日

![](images/improved-user-experience-for-vercel-documentation-vercel/img_001.jpg)![](images/improved-user-experience-for-vercel-documentation-vercel/img_002.jpg)

We've redesigned and improved the Vercel documentation with:

我们已对 Vercel 文档进行了重新设计与优化，主要改进包括：

- **Updated navigation**: Navigation is now separated by product categories. You can quickly view all products in a category by hovering the navigation item.

- **导航更新**：导航现已按产品类别划分。将鼠标悬停在导航项上，即可快速查看该类别下的全部产品。

- **Customization**: You can use the global frameworks toggle to show code examples with your favorite framework.

- **自定义功能**：您可通过全局框架切换器，显示您最喜爱的框架对应的代码示例。

- **All Products page**: You can now see [all Vercel products](https://vercel.com/docs/all-products) on a single documentation page.

- **“全部产品”页面**：您现在可在单个文档页面中浏览 [Vercel 全部产品](https://vercel.com/docs/all-products)。

- **Improved mobile design:** The new mobile-friendly navigation enables you to discover and read easily when you’re on the go.

- **移动端体验优化**：全新适配移动设备的导航设计，让您在移动场景下也能轻松发现内容并流畅阅读。

[Get started](https://vercel.com/docs) with the Vercel documentation today.

立即 [开始使用](https://vercel.com/docs) Vercel 文档。
