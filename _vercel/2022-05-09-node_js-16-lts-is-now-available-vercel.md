2022年5月9日

![](images/node_js-16-lts-is-now-available-vercel/img_001.jpg)![](images/node_js-16-lts-is-now-available-vercel/img_002.jpg)

As of today, version 16 of Node.js can be selected in the **Node.js Version** section on the General page in the **Project Settings** (newly created Projects will default to the new version).

自今日起，您可在 **Project Settings**（项目设置）的通用（General）页面中的 **Node.js Version**（Node.js 版本）选项中选择 Node.js 16 版本（新创建的项目将默认使用该版本）。

The new version introduces several [new features](https://medium.com/the-node-js-collection/node-js-16-available-now-7f5099a97e70) including:

新版本引入了多项[新特性](https://medium.com/the-node-js-collection/node-js-16-available-now-7f5099a97e70)，包括：

- ECMAScript RegExp Match Indices  
- ECMAScript 正则表达式匹配索引（RegExp Match Indices）

- `AbortController`  
- `AbortController`

- `AggregateError`  
- `AggregateError`

- `Array.prototype.at()`  
- `Array.prototype.at()`

- `require('crypto').webcrypto`  
- `require('crypto').webcrypto`

- `require('timers/promises')`  
- `require('timers/promises')`

- `fs.cp()`

- `fs.cp()`

The exact version used today is [16.14.0](https://github.com/nodejs/node/blob/master/doc/changelogs/CHANGELOG_V16.md#16.14.0) and will automatically update minor and patch releases. Therefore, only the major version (16.x) is guaranteed.

当前实际使用的版本为 [16.14.0](https://github.com/nodejs/node/blob/master/doc/changelogs/CHANGELOG_V16.md#16.14.0)，且会自动更新次要版本（minor）和修订版本（patch）。因此，仅主版本号（16.x）可确保稳定。

Check out [the documentation](https://vercel.com/docs/runtimes#official-runtimes/node-js/node-js-version) as well.

也请参阅 [相关文档](https://vercel.com/docs/runtimes#official-runtimes/node-js/node-js-version)。
