Vercel [Web Analytics](https://vercel.com/changelog/web-analytics-is-now-generally-available) 现已面向 Pro 和 Enterprise 用户开放自定义事件（Custom Events）追踪功能，支持在前端应用中进行集成。借助自定义事件，您可以精准衡量用户行为，例如新闻通讯订阅情况，或分析客户最常点击的各类行动号召（CTA），从而更深入地理解用户的使用路径与行为模式。

自定义事件功能已包含在 Pro 与 Enterprise 套餐中。Pro 用户每个自定义事件对象最多可追踪 2 个键（key）；Enterprise 用户则可根据需要自定义配置键的数量。此外，Pro 用户还可按每月 50 美元的价格额外订购 [Web Analytics Plus](https://vercel.com/docs/concepts/analytics/limits-and-pricing)，以将单个事件对象支持的键数量提升至 8 个。

如需开始追踪自定义事件，请将我们的 npm 包 `@vercel/analytics` 升级至最新版 `1.0.0`。

[查阅我们的官方文档](https://vercel.com/docs/concepts/analytics/custom-events)，了解更多详细信息。
