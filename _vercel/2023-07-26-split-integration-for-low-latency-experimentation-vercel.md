2023年7月26日

![](images/split-integration-for-low-latency-experimentation-vercel/img_001.jpg)![](images/split-integration-for-low-latency-experimentation-vercel/img_002.jpg)

The Split integration syncs the feature flags you have in Split already into [Vercel Edge Config](https://vercel.com/docs/storage/edge-config) to help safely launch your releases and experiments.

Split 集成可将您已在 Split 中配置的功能标志（feature flags）同步至 [Vercel Edge Config](https://vercel.com/docs/storage/edge-config)，从而帮助您更安全地发布新功能与开展实验。

With near-zero latency storage provided by Vercel Edge Config your feature flags are immediately available to SDKs within the Vercel network. This improves performance and load experience when deploying features and experiments; all while Split keeps your data up-to-date.

得益于 Vercel Edge Config 提供的近乎零延迟存储能力，您的功能标志可即时被 Vercel 网络内的 SDK 所获取。这显著提升了功能与实验上线时的性能表现和加载体验；与此同时，Split 持续确保您的数据保持最新状态。

This integration is available to users on all plans.

该集成面向所有订阅计划的用户开放。

Check out the [integration](https://vercel.com/integrations/split) to get started.

请访问 [该集成页面](https://vercel.com/integrations/split) 开始使用。
