Dec 14, 2021

2021年12月14日

React is one of the most popular ways to build user interfaces. Many of the world's largest enterprises and newest startups are building their online presence with it, pushing demand for React developers, improvements to React, and learning resources to an all-time high.

React 是构建用户界面最流行的方式之一。全球众多顶尖企业及新兴初创公司均采用 React 构建其线上业务，由此推动了市场对 React 开发者、React 自身功能改进以及相关学习资源的空前需求。

Since 2016, Vercel has worked closely with the React core team at Meta, as well as hired contributors and key members of the React community to accelerate mutual goals. Supporting open-source projects is an [integral part of our mission](https://vercel.com/blog/vercel-funding-series-d-and-valuation#support-open-source-projects) to "Make the Web. Faster." and we want to help React continue to be successful now and in the future.

自 2016 年起，Vercel 便与 Meta 的 React 核心团队保持紧密合作，并聘请了多位 React 社区的贡献者及关键成员，以加速双方共同目标的实现。支持开源项目是我们“让网络更快（Make the Web. Faster.）”这一使命中[不可或缺的一环](https://vercel.com/blog/vercel-funding-series-d-and-valuation#support-open-source-projects)，我们亦希望助力 React 在当下及未来持续取得成功。

[Sebastian Markbåge](https://twitter.com/sebmarkbage) on the React core team at Meta is [joining Vercel next month](https://twitter.com/sebmarkbage/status/1470761449136001027). As part of his role at Vercel, he'll still provide leadership on the React core team and help maintain the direction of React. Vercel will continue to work on adopting React 18 and [React Server Components](https://vercel.com/blog/everything-about-react-server-components) into Next.js, and contribute to future React [working groups](https://github.com/reactwg).

Meta 公司 React 核心团队成员 [Sebastian Markbåge](https://twitter.com/sebmarkbage) 将于下月[加入 Vercel](https://twitter.com/sebmarkbage/status/1470761449136001027)。在 Vercel 任职期间，他将继续在 React 核心团队中担任领导角色，并协助把握 React 的整体发展方向。Vercel 将持续推进 React 18 和 [React Server Components（React 服务端组件）](https://vercel.com/blog/everything-about-react-server-components) 在 Next.js 中的集成工作，并积极参与未来 React 各类 [工作组（working groups）](https://github.com/reactwg) 的建设与贡献。

The Next.js team is particularly excited to work more directly with Sebastian. Vercel will continue to invest in other OSS and support the communities our customers depend on – including React, [Next.js](http://nextjs.org/), [Svelte](https://vercel.com/blog/vercel-welcomes-rich-harris-creator-of-svelte), [Turborepo](https://vercel.com/blog/vercel-acquires-turborepo), [SWC](https://twitter.com/kdy1dev/status/1417842357416824833), [Webpack](https://twitter.com/wsokra/status/1381860800533528576), [SWR](https://twitter.com/shuding_/status/1465710767865401344), and many more.

Next.js 团队尤其期待能更直接地与 Sebastian 展开合作。Vercel 将持续投入其他开源项目（OSS），并支持我们的客户所依赖的各类社区——包括 React、[Next.js](http://nextjs.org/)、[Svelte](https://vercel.com/blog/vercel-welcomes-rich-harris-creator-of-svelte)、[Turborepo](https://vercel.com/blog/vercel-acquires-turborepo)、[SWC](https://twitter.com/kdy1dev/status/1417842357416824833)、[Webpack](https://twitter.com/wsokra/status/1381860800533528576)、[SWR](https://twitter.com/shuding_/status/1465710767865401344)，以及更多项目。
