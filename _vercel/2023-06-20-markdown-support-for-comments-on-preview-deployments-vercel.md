借助在预览部署（Preview Deployments）上[添加评论](https://vercel.com/docs/workflow-collaboration/comments/using-comments)的功能，任何被添加到您项目中的协作者均可直接针对文案、组件及交互行为发表评论。如今，评论功能已支持 Markdown 语法，您可使用列表、粗体、链接、引用等多种格式来丰富您的评论内容。

您可通过以下行内字符快捷触发相应格式：

- `*` 表示粗体（或按 Ctrl/Cmd+B）

- `_` 表示斜体（或按 Ctrl/Cmd+I）

- `~` 表示删除线（或按 Ctrl/Cmd+Shift+X）

- ````` 表示行内代码（或按 Ctrl/Cmd+E）

- `>` 加空格表示引用块开头

- `-` 或 `*` 或 `1.` 加空格表示列表项开头

- `Tab` 或 `Shift+Tab` 用于调整缩进

- `Tab` 或 `Shift+Tab` 用于调整缩进

工具栏还将包含一个用于基本行内样式的按钮；如果单击某个链接，我们将弹出一个新窗口，用于编辑链接文本和 URL。
