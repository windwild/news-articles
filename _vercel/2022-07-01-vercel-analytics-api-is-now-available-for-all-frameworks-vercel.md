[Vercel Analytics](https://vercel.com/analytics)

Vercel Analytics helps you understand the performance of your application based on real visitor data. With the Vercel Analytics API, you can now use Vercel Analytics with any framework.

Vercel Analytics 可帮助您基于真实访客数据了解应用的性能表现。借助 Vercel Analytics API，您现在可在任意框架中使用 Vercel Analytics。

We currently have zero-configuration Analytics support for Next.js, Nuxt, and Gatsby. Now, any website can monitor its Core Web Vitals with Vercel Analytics by using the API directly. Get started today with the [SvelteKit](https://github.com/vercel/vercel/tree/main/examples/sveltekit) or [Create React App](https://github.com/vercel/vercel/tree/main/examples/create-react-app) starts, which have been updated to include support for the Vercel Analytics API.

目前，我们已为 Next.js、Nuxt 和 Gatsby 提供开箱即用（零配置）的 Analytics 支持。如今，任何网站均可直接调用该 API，通过 Vercel Analytics 监控其核心网页指标（Core Web Vitals）。立即开始使用已更新支持 Vercel Analytics API 的 [SvelteKit](https://github.com/vercel/vercel/tree/main/examples/sveltekit) 或 [Create React App](https://github.com/vercel/vercel/tree/main/examples/create-react-app) 示例项目。

[Check out the documentation](https://vercel.com/docs/concepts/analytics/api) as well.  

也请参阅相关[文档](https://vercel.com/docs/concepts/analytics/api)。
