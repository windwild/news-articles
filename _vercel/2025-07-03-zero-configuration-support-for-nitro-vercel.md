2025年7月3日

![](images/zero-configuration-support-for-nitro-vercel/img_001.jpg)![](images/zero-configuration-support-for-nitro-vercel/img_002.jpg)

Vercel now supports [Nitro](https://nitro.build/) applications, a backend toolkit for building web servers, with zero-configuration.

Vercel 现已支持 [Nitro](https://nitro.build/) 应用——一款用于构建 Web 服务器的后端工具包，开箱即用，无需配置。

Nitro powers frameworks like Nuxt.js, TanStack Start, and SolidStart.

Nitro 为 Nuxt.js、TanStack Start 和 SolidStart 等框架提供底层支持。

[Deploy Nitro on Vercel](https://vercel.com/new/clone?repository-url=https://github.com/vercel/vercel/tree/main/examples/nitro) or [visit Nitro's Vercel documentation](https://nitro.build/deploy/providers/vercel).

[在 Vercel 上部署 Nitro](https://vercel.com/new/clone?repository-url=https://github.com/vercel/vercel/tree/main/examples/nitro) 或 [访问 Nitro 的 Vercel 文档](https://nitro.build/deploy/providers/vercel)。
