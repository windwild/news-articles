Nov 20, 2020

2020 年 11 月 20 日

![](images/projects-can-now-be-renamed-vercel/img_001.jpg)

对于您过去在 Vercel 上部署的所有 Project，平台会根据您的 Git 仓库名称或本地目录名称自动为您选定一个名称；或者，您也可以在创建 Project 前手动自定义其名称。

此前，Project 创建完成后无法再更改其名称。但从今天起，您可直接在 Project 设置中的“General”（常规）页面修改 Project 名称。

在更改 Project 名称时，无论您是通过 [Git 仓库进行部署](https://vercel.com/docs/git)，还是使用 Vercel CLI [将 Project 关联至本地目录](https://vercel.com/docs/cli#commands/overview/project-linking)，您本人或团队的工作流程均不会受到任何影响。

您也可查阅 [相关文档](https://vercel.com/docs/platform/projects#project-name)。
