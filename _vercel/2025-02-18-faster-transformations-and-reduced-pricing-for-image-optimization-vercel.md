Feb 18, 2025

2025年2月18日

![](images/faster-transformations-and-reduced-pricing-for-image-optimization-vercel/img_001.jpg)![](images/faster-transformations-and-reduced-pricing-for-image-optimization-vercel/img_002.jpg)

We’ve optimized our Image Optimization infrastructure, including:

我们已优化了图像优化（Image Optimization）基础设施，包括：

- **60% faster** transformations

- **转换速度提升 60%**

- New, opt-in reduced pricing

- 新增可选的降价方案

Previously, usage was measured by the number of unique source images ($5 per 1K source images). You can now opt into usage based on transformations with regional pricing, starting from:

此前，用量按唯一源图像数量计费（每 1,000 张源图像 $5）。现在，您可选择按图像转换操作（transformations）计费，并采用区域化定价，起始价格如下：

- **Image Transformations:** $0.05 per 1K image transformations

- **图像转换操作：** 每 1,000 次图像转换 $0.05

- **Image Cache Reads:** $0.40 per 1M cache read units

- **图像缓存读取：** 每 100 万次缓存读取单位 $0.40

- **Image Cache Writes:** $4.00 per 1M cache write units

- **图像缓存写入：** 每 100 万次缓存写入单位 $4.00

This new pricing model is **opt-in** through your project settings.

该新定价模式需通过项目设置**主动启用（opt-in）**。

- There are **no changes to existing customers** using Image Optimization  
- 使用图像优化服务的**现有客户不会发生任何变更**

- New projects for existing customers will also have no changes  
- 现有客户的新增项目同样不会发生任何变更

- New customers will start on the new pricing today  
- 新客户将从今日起适用新的定价方案

Pro and self-serve Enterprise customers can view the projected cost difference when [enabling in settings](https://vercel.com/d?to=%2F%5Bteam%5D%2F~%2Fsettings%2Fbilling%23image-optimization-new-price&title=Go+to+Billing+Settings). All Enterprise customers can also reach out to their account team to discuss new pricing.  
Pro 用户及自助式 Enterprise 用户可在[设置中启用该功能](https://vercel.com/d?to=%2F%5Bteam%5D%2F~%2Fsettings%2Fbilling%23image-optimization-new-price&title=Go+to+Billing+Settings)时查看预估费用差异。所有 Enterprise 用户亦可联系其客户成功团队，进一步探讨新定价方案。

Hobby customers have been moved to the new model's included allotments.  
Hobby 用户已迁移至新计费模型所包含的配额体系。

Learn more about [Image Optimization pricing](https://vercel.com/docs/image-optimization/limits-and-pricing).  
了解更多关于[图像优化定价](https://vercel.com/docs/image-optimization/limits-and-pricing)的信息。
