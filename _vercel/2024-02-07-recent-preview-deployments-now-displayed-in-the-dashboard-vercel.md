![](images/recent-preview-deployments-now-displayed-in-the-dashboard-vercel/img_001.jpg)![](images/recent-preview-deployments-now-displayed-in-the-dashboard-vercel/img_002.jpg)

您最近查看或部署的预览（Preview）现在可在仪表板（Dashboard）的“近期预览”（Recent Previews）列中直接访问。

每个近期预览条目均包含指向 Vercel 上该部署页面的链接；当可用时，还包含指向代码托管平台（如 GitHub、GitLab 等）上对应 Pull Request 或源代码的链接。

了解更多，请参阅 [仪表板概览](https://vercel.com/docs/dashboard-features) 文档。
