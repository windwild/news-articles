Oct 12, 2022

2022年10月12日

![](images/explore-bot-traffic-data-now-in-monitoring-beta-vercel/img_001.jpg)![](images/explore-bot-traffic-data-now-in-monitoring-beta-vercel/img_002.jpg)

Monitoring now lets you explore traffic data that comes from known and unknown bots.  
监控功能现已支持探索来自已知与未知机器人的流量数据。

You can group the traffic data by `public_ip`, `user_agent`, `asn`, and `bot_name` to efficiently debug issues related to traffic coming from real users or bots.  
您可按 `public_ip`、`user_agent`、`asn` 和 `bot_name` 对流量数据进行分组，从而高效地排查源自真实用户或机器人的流量相关问题。

Three new example queries have been added to help you get started:  
我们新增了三个示例查询，助您快速上手：

1. Requests by IP Address  
1. 按 IP 地址统计的请求

2. Requests by Bot/Crawler  
2. 按机器人/爬虫统计的请求

3. Requests by User Agent  
3. 按 User Agent 统计的请求

[Check out the documentation](https://vercel.com/docs/concepts/dashboard-features/monitoring) to learn more.  
[查看文档](https://vercel.com/docs/concepts/dashboard-features/monitoring) 了解更多信息。
