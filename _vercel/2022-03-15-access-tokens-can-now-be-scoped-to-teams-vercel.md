现在，CLI 中使用的访问令牌以及用于 API 身份验证的访问令牌，均可限定为特定团队（Team）范围。

此项改进为通过 Vercel CLI 或 API 扩展 Vercel 平台的用户提供了更强的安全性与更精细的权限控制。

也欢迎查阅[相关文档](https://vercel.com/docs/rest-api#introduction/api-basics/authentication)。
